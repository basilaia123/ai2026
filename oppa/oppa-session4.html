💼 HR Case — CV Screening & Multimodal Extraction

ხედვის AI იდეალურად მუშაობს CV-ების (სურათების, PDF-ების) სკანირებისას. შეგვიძლია რამდენიმე წამში ამოვიღოთ სტრუქტურირებული ინფორმაცია და შევადაროთ ვაკანსიის მოთხოვნებს.

CV-ის სკრინინგი და შეფასება ვაკანსიის მიხედვით

გაანალიზე თანდართული CV (სურათი/PDF) და შეადარე იგი OPPA-ს ვაკანსიის მოთხოვნებს:
[აქ ჩაწერეთ ვაკანსიის მოთხოვნები, მაგ. Customer Support Specialist]

გთხოვ დააბრუნო პასუხი შემდეგ სტრუქტურაში:
1. Candidate Summary: (მოკლე აღწერა)
2. key_skills_matched: (თანხვედრი უნარები)
3. gaps_identified: (აკლებული უნარები ან გამოცდილება)
4. Score: (ქულა 1-10 სკალაზე, დაასაბუთე რატომ)
5. Recommendation: (მივიღოთ თუ არა გასაუბრებაზე - YES/NO/MAYBE)

⚙️ როგორ მუშაობს Advanced Data Analysis (Python Runtime Sandbox)?

როდესაც ატვირთავთ Excel/CSV ფაილს, AI მოდელი უბრალოდ კი არ კითხულობს მონაცემებს ტექსტურად, არამედ წერს და უშვებს Python-ის კოდს დაცულ გარემოში (Sandbox). ეს უზრუნველყოფს 100%-ით ზუსტ მათემატიკურ გამოთვლებს და გამორიცხავს ჰალუცინაციებს.

1. სტრუქტურის გაგება AI კითხულობს სვეტების სათაურებს და მონაცემთა ტიპებს.
2. Python-ის წერა ავტომატურად წერს Pandas/Numpy კოდს ფაილის დასამუშავებლად.
3. კოდის შესრულება ასრულებს კოდს Sandbox-ში და იღებს ზუსტ გამოთვლებს.
4. პასუხის ჩვენება გამოთვლების შედეგები გადმოაქვს გრაფიკში, ცხრილში ან ტექსტში.
💡 გამოდგება აქ Gemini და Claude? მათაც აქვთ Sandbox?

დიახ, 2026 წლისთვის სამივე წამყვან პლატფორმას (ChatGPT, Google Gemini და Anthropic Claude) აქვს საკუთარი დაცული გარემო (Sandbox) მონაცემთა კოდირებული ანალიზისთვის:

🟢 ChatGPT (Advanced Data Analysis) ფლაგმანი Python Sandbox (ყოფილი Code Interpreter). წერს და უშვებს Python-ის კოდს, აშენებს გრაფიკებს, აკონვერტირებს და წერს ფაილებს, აკეთებს zip/unzip-ს.
🔵 Google Gemini (Python Code Execution) Gemini-ს აქვს ჩაშენებული Python კოდის გაშვების გარემო Google-ის სერვერებზე. რთული მათემატიკური ანალიზისას მყისიერად უშვებს კოდს 100%-იანი სიზუსტისთვის.
🟠 Anthropic Claude (Analysis Tool) Claude-ში ჩაშენებულია **Analysis Tool** (ვიზუალური Sandbox). ფაილის ატვირთვისას ის უშვებს JavaScript/Python კოდს, ითვლის ზუსტ შედეგებს და აგებს ინტერაქტიულ გრაფიკებს.

Claude Code-ის ლოკალური სკრიპტინგის პრომპტი (Schema-Only)

Write a Python script that reconciles two files locally. Do not send or upload the actual data rows.
Here is only the schema (metadata):
- ledger.csv columns: [ref_id, date, amount, status]
- bank_statement.csv columns: [bank_ref, date, amount, type]

Requirements:
1. Use Pandas to load both CSV files locally.
2. Perform an outer join on ref_id (ledger) vs bank_ref (bank_statement).

📊 Gemini for Google Sheets — მონაცემთა შევსება და ავტომატიზაცია (June 2026 Update)

💡 გამოყენების სცენარი OPPA-სთვის: მერჩანტებისა და კლიენტების მონაცემთა ბაზების გასუფთავება (არასტანდარტული მისამართები, ტელეფონის ნომრები, ორთოგრაფიული შეცდომები), საუბრების სენტიმენტ-ანალიზი და მხარდაჭერის ბილეთების ავტომატური კლასიფიკაცია.

📋 ძირითადი AI ფორმულები Google Sheets-ში:
