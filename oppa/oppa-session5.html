  • рҹ§ұ бғӣбғқбғ“бғЈбғҡбғҳ (Module): бғЎбғӘбғ”бғңбғҗбғ бғҳбғЎ бғӘбғҗбғҡбғҷбғ”бғЈбғҡбғҳ бғЁбғ”бғӣбғҗбғ“бғ’бғ”бғңбғ”бғҡбғҳ бғңбғҗбғ‘бғҳбғҜбғҳ (бғӣбғҗбғ’. Google Sheets, HTTP Module, Slack), бғ бғқбғӣбғ”бғҡбғҳбғӘ бғҗбғЎбғ бғЈбғҡбғ”бғ‘бғЎ бғҷбғқбғңбғҷбғ бғ”бғўбғЈбғҡ бғҘбғӣбғ”бғ“бғ”бғ‘бғҗбғЎ.
  • вҸұ бғўбғ бғҳбғ’бғ”бғ бғҳ (Trigger): бғӣбғқбғ•бғҡбғ”бғңбғҗ, бғ бғқбғӣбғ”бғҡбғҳбғӘ бғҳбғ¬бғ§бғ”бғ‘бғЎ бғЎбғӘбғ”бғңбғҗбғ бғЎ. бғҗбғ бғЎбғ”бғ‘бғқбғ‘бғЎ Polling (бғһбғ”бғ бғҳбғқбғ“бғЈбғҡбғҳ бғЁбғ”бғӣбғқбғ¬бғӣбғ”бғ‘бғҗ, бғӣбғҗбғ’. 15 бғ¬бғЈбғ—бғЁбғҳ бғ”бғ бғ—бғ®бғ”бғҡ) бғ“бғҗ Instant/Webhook (бғӣбғ§бғҳбғЎбғҳбғ”бғ бғҳ бғ бғ”бғҗбғ’бғҳбғ бғ”бғ‘бғҗ бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғҳбғЎ бғЁбғ”бғӣбғқбғЎбғ•бғҡбғҳбғЎбғ—бғҗбғңбғҗбғ•бғ”).
  • вҡҷпёҸ бғӣбғқбғҘбғӣбғ”бғ“бғ”бғ‘бғҗ (Action): бғӣбғқбғ“бғЈбғҡбғҳ, бғ бғқбғӣбғ”бғҡбғҳбғӘ бғҗбғЎбғ бғЈбғҡбғ”бғ‘бғЎ бғ“бғҗбғ•бғҗбғҡбғ”бғ‘бғҗбғЎ (бғӣбғҗбғ’. бғ”бғҡ-бғӨбғқбғЎбғўбғҳбғЎ бғ’бғҗбғ’бғ–бғҗбғ•бғңбғҗ, бғҗбғ®бғҗбғҡбғҳ бғЎбғўбғ бғҳбғҘбғқбғңбғҳбғЎ бғ©бғҗбғ¬бғ”бғ бғҗ).
  • рҹ”Җ бғ бғқбғЈбғўбғ”бғ бғҳ (Router): бғ§бғқбғӨбғЎ бғЎбғӘбғ”бғңбғҗбғ бғЎ бғ бғҗбғӣбғ“бғ”бғңбғҳбғӣбғ” бғһбғҗбғ бғҗбғҡбғ”бғҡбғЈбғ  бғўбғқбғўбғҗбғ“ бғһбғҳбғ бғқбғ‘бғ”бғ‘бғҳбғЎ бғӣбғҳбғ®бғ”бғ“бғ•бғҳбғ—.
  • рҹ”Қ бғӨбғҳбғҡбғўбғ бғҳ (Filter): бғ–бғҰбғЈбғ“бғҗбғ•бғЎ бғӣбғқбғңбғҗбғӘбғ”бғӣбғ—бғҗ бғңбғҗбғҷбғҗбғ“бғЎ бғ“бғҗ бғҗбғўбғҗбғ бғ”бғ‘бғЎ бғӣбғ®бғқбғҡб

    рҹҡҖ 6 бғһбғқбғўбғ”бғңбғӘбғҳбғЈбғ бғҳ Make.com / n8n бғЎбғӘбғ”бғңбғҗбғ бғҳ OPPA-бғЎбғ—бғ•бғҳбғЎ

    бғҗбғҘ бғӣбғқбғӘбғ”бғӣбғЈбғҡбғҳбғҗ 6 бғ бғ”бғҗбғҡбғЈбғ бғҳ бғ‘бғҳбғ–бғңбғ”бғЎ-бғҷбғ”бғҳбғЎбғҳ, бғЎбғҗбғ“бғҗбғӘ OPPA-бғЎ бғЁбғ”бғЈбғ«бғҡбғҳбғҗ бғ’бғҗбғӣбғқбғҳбғ§бғ”бғңбғқбғЎ Make.com бғҗбғң n8n бғЎбғҗбғӣбғЈбғЁбғҗбғқ бғһбғ бғқбғӘбғ”бғЎбғ”бғ‘бғҳбғЎ бғқбғһбғўбғҳбғӣбғҳбғ–бғҗбғӘбғҳбғҳбғЎбғ—бғ•бғҳбғЎ:

    1. PayBox Kiosk бғҳбғңбғҷбғҗбғЎбғҗбғӘбғҳбғҳбғЎ бғқбғһбғўбғҳбғӣбғҳбғ–бғҗбғӘбғҳбғҗ (бғҷбғ бғҳбғўбғҳбғҷбғЈбғҡбғҳ бғЁбғ”бғ•бғЎбғ”бғ‘бғҗ)

    бғўбғ бғҳбғ’бғ”бғ бғҳ: PayBox-бғҳбғЎ бғ‘бғҗбғңбғҷбғңбғқбғўбғҳбғЎ бғӣбғҳбғӣбғҰбғ”бғ‘бғҳ (Cash Acceptor) бғҳбғ•бғЎбғ”бғ‘бғҗ 90%-бғҳбғ— вһ” бғҳбғ’бғ–бғҗбғ•бғңбғ”бғ‘бғҗ бғӣбғ§бғҳбғЎбғҳбғ”бғ бғҳ Webhook (n8n/Make).

    AI/бғҡбғқбғ’бғҳбғҷбғҗ: бғЎбғҳбғЎбғўбғ”бғӣбғҗ бғҗбғҜбғ’бғЈбғӨбғ”бғ‘бғЎ бғҗбғ®бғҡбғқбғӣбғ“бғ”бғ‘бғҗбғ бғ” бғЎбғ®бғ•бғҗ бғЁбғ”бғ•бғЎбғ”бғ‘бғЈбғҡ бғҗбғһбғҗбғ бғҗбғўбғ”бғ‘бғЎ, Google Maps API-бғҳбғ— бғҗбғ“бғ’бғ”бғңбғЎ бғқбғһбғўбғҳбғӣбғҗбғҡбғЈбғ  бғЎбғҗбғӣбғҗбғ бғЁбғ бғЈбғўбғқ бғ бғЈбғҷбғҗбғЎ бғҳбғңбғҷбғҗбғЎбғҗбғӘбғҳбғҳбғЎбғ—бғ•бғҳбғЎ.

    бғЁбғ”бғ“бғ”бғ’бғҳ: SMS/Slack бғЁбғ”бғўбғ§бғқбғ‘бғҳбғңбғ”бғ‘бғҗ бғҳбғңбғҷбғҗбғЎбғҗбғўбғқбғ бғ—бғҗбғң бғ“бғҗ бғқбғһбғўбғҳбғӣбғҳбғ–бғ”бғ‘бғЈбғҡбғҳ бғӣбғҗбғ бғЁбғ бғЈбғўбғҳ, бғ бғҗбғӘ бғҗбғӣбғӘбғҳбғ бғ”бғ‘бғЎ бғўбғ бғҗбғңбғЎбғһбғқбғ бғўбғҳбғ бғ”бғ‘бғҳбғЎ бғ®бғҗбғ бғҜбғЎ бғ“бғҗ бғҗбғһбғҗбғ бғҗбғўбғ”бғ‘бғҳбғЎ бғ’бғҗбғ©бғ”бғ бғ”бғ‘бғҳбғЎ бғ“бғ бғқбғЎ.

    2. бғўбғ бғҗбғңбғ–бғҗбғҘбғӘбғҳбғ”бғ‘бғҳбғЎ бғ§бғқбғ•бғ”бғҡбғ“бғҰбғҳбғЈбғ бғҳ бғҗбғ•бғўбғқбғӣбғҗбғўбғЈбғ бғҳ Reconciliation (бғ‘бғҳбғҡбғҳбғңбғ’бғҳ)

    бғўбғ бғҳбғ’бғ”бғ бғҳ: Schedule (бғ§бғқбғ•бғ”бғҡ бғҰбғҗбғӣбғ” 02:00 бғЎбғҗбғҗбғ—бғ–бғ”) вһ” n8n/Make бғЈбғҷбғҗбғ•бғЁбғҳбғ бғ“бғ”бғ‘бғҗ OPPA Core бғ‘бғҗбғ–бғҗбғЎ бғ“бғҗ бғһбғҗбғ бғўбғңбғҳбғқбғ бғҳ бғ‘бғҗбғңбғҷбғ”бғ‘бғҳбғЎ API-бғ”бғ‘бғЎ.

    AI/бғҡбғқбғ’бғҳбғҷбғҗ: бғҗбғ“бғҗбғ бғ”бғ‘бғЎ OPPA-бғЎ бғ‘бғҳбғҡбғҳбғ’бғҳбғЎ бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғЎ бғ‘бғҗбғңбғҷбғҳбғЎ бғ бғ”бғҗбғҡбғЈбғ  бғҗбғӣбғқбғңбғҗбғ¬бғ”бғ бғ”бғ‘бғ—бғҗбғң. бғЁбғ”бғЈбғЎбғҗбғ‘бғҗбғӣбғқбғ‘бғҳбғЎ (Outlier) бғҗбғҰбғӣбғқбғ©бғ”бғңбғҳбғЎбғҗбғЎ бғ©бғҗбғҳбғ бғ—бғ•бғ”бғ‘бғҗ Router.

    бғЁбғ”бғ“бғ”бғ’бғҳ: бғЁбғ”бғЈбғЎбғҗбғ‘бғҗбғӣбғқ бғўбғ бғҗбғңбғ–бғҗбғҘбғӘбғҳбғ”бғ‘бғҳ бғҗбғ•бғўбғқбғӣбғҗбғўбғЈбғ бғҗбғ“ бғҳбғ’бғ–бғҗбғ•бғңбғ”бғ‘бғҗ Accounting-бғҳбғЎ Slack-бғҗбғ бғ®бғЁбғҳ бғўбғҳбғҷбғ”бғўбғҳбғЎ бғЎбғҗбғ®бғҳбғ—, бғ бғҗбғӘ бғ’бғҗбғӣбғқбғ бғҳбғӘбғ®бғҗбғ•бғЎ бғ®бғ”бғҡбғҳбғ— бғЁбғ”бғӣбғқбғ¬бғӣбғ”бғ‘бғҗбғЎ.

    3. PayBox бғҘбғЎбғ”бғҡбғҳбғЎ бғӣбғқбғңбғҳбғўбғқбғ бғҳбғңбғ’бғҳ бғ“бғҗ бғҗбғҰбғ“бғ’бғ”бғңбғҳбғЎ бғЎбғҷбғ бғҳбғһбғўбғҳ (Auto-Reboot)

    бғўбғ бғҳбғ’бғ”бғ бғҳ: бғўбғ”бғ бғӣбғҳбғңбғҗбғҡбғҳ бғ’бғҗбғ“бғҳбғЎ бғқбғӨбғҡбғҗбғҳбғңбғЁбғҳ вһ” Make/n8n бғҳбғҰбғ”бғ‘бғЎ бғӣбғ§бғҳбғЎбғҳбғ”бғ  Alert-бғЎ бғҘбғЎбғ”бғҡбғҳбғЎ бғӣбғқбғңбғҳбғўбғқбғ бғҳбғңбғ’бғҳбғЎ бғЎбғҳбғЎбғўбғ”бғӣбғҳбғ“бғҗбғң.

    AI/бғҡбғқбғ’бғҳбғҷбғҗ: n8n HTTP API-бғҳбғ— бғЈбғ’бғ–бғҗбғ•бғңбғҳбғЎ бғ’бғҗбғ“бғҗбғўбғ•бғҳбғ бғ—бғ•бғҳбғЎ (Reboot) бғ‘бғ бғ«бғҗбғңбғ”бғ‘бғҗбғЎ бғЁбғ”бғЎбғҗбғ‘бғҗбғӣбғҳбғЎ GPRS/LTE бғ бғқбғЈбғўбғ”бғ бғЎ. бғ”бғҡбғқбғ“бғ”бғ‘бғҗ 5 бғ¬бғЈбғ—бғЎ бғ“бғҗ бғҷбғ•бғҡбғҗбғ• бғҗбғӣбғқбғ¬бғӣбғ”бғ‘бғЎ бғһбғҳбғңбғ’бғЎ.

    бғЁбғ”бғ“бғ”бғ’бғҳ: бғ—бғЈ бғҷбғҗбғ•бғЁбғҳбғ бғҳ бғҗбғҰбғ“бғ’бғҗ, бғўбғҳбғҷбғ”бғўбғҳ бғҳбғ®бғЈбғ бғ”бғ‘бғҗ; бғ—бғЈ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗ бғ“бғҗбғ бғ©бғҗ, бғҗбғ•бғўбғқбғӣбғҗбғўбғЈбғ бғҗбғ“ бғҳбғ®бғЎбғңбғ”бғ‘бғҗ бғ‘бғҗбғ бғҗбғ—бғҳ Jira Service Desk-бғЁбғҳ бғ“бғҗ бғ’бғҗбғ“бғҗбғ”бғӘбғ”бғӣбғҗ бғўбғ”бғҘбғңбғҳбғҷбғқбғЎбғЎ.

    4. бғӣбғқбғӣбғ®бғӣбғҗбғ бғ”бғ‘бғ”бғҡбғ—бғҗ Viber/Telegram бғ©бғҗбғўбғ‘бғқбғўбғҳ CRM бғЎбғҳбғңбғҘбғ бғқбғңбғҳбғ–бғҗбғӘбғҳбғҳбғ—

    бғўбғ бғҳбғ’бғ”бғ бғҳ: бғҷбғҡбғҳбғ”бғңбғўбғҳ Viber/Telegram бғ‘бғқбғўбғҳбғ“бғҗбғң бғҗбғ’бғ–бғҗбғ•бғңбғҳбғЎ бғЁбғ”бғўбғ§бғқбғ‘бғҳбғңбғ”бғ‘бғҗбғЎ бғ©бғҗбғ бғҳбғӘбғ®бғ•бғҳбғЎ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗбғ–бғ”.

    AI/бғҡбғқбғ’бғҳбғҷбғҗ: Claude API бғҳбғҰбғ”бғ‘бғЎ бғЁбғ”бғўбғ§бғқбғ‘бғҳбғңбғ”бғ‘бғҗбғЎ, бғҗбғӣбғқбғҳбғӘбғңбғқбғ‘бғЎ бғўбғ”бғ бғӣбғҳбғңбғҗбғҡбғҳбғЎ ID-бғЎ бғ“бғҗ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗбғЎ, бғҗбғӣбғқбғ¬бғӣбғ”бғ‘бғЎ бғҷбғҡбғҳбғ”бғңбғўбғҳбғЎ бғ‘бғҗбғҡбғҗбғңбғЎбғЎ бғ“бғҗ бғҗбғӣбғ–бғҗбғ“бғ”бғ‘бғЎ бғһбғҗбғЎбғЈбғ®бғЎ.

    AI/бғҡбғқбғ’бғҳбғҷбғҗ: бғҗбғ’бғ бғқбғ•бғ”бғ‘бғЎ бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғЎ PEP (бғһбғқбғҡбғҳбғўбғҳбғҷбғЈбғ бғҗбғ“ бғҗбғҘбғўбғҳбғЈбғ бғҳ) бғ“бғҗ бғӣбғҗбғҰбғҗбғҡбғҳ бғ бғҳбғЎбғҷбғҳбғЎ бғӣбғ”бғ бғ©бғҗбғңбғўбғ”бғ‘бғ–бғ”, бғҗбғӨбғҳбғҡбғўбғ бғҗбғ•бғЎ PII бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғЎ бғҡбғқбғҷбғҗбғҡбғЈбғ бғҗбғ“ бғ“бғҗ бғҗбғ¬бғ§бғқбғ‘бғЎ NBG-бғҳбғЎ бғЁбғҗбғ‘бғҡбғқбғңбғҳбғЎ бғӣбғҳбғ®бғ”бғ“бғ•бғҳбғ—.

NCR бғҗбғң Wincor).

3. PayBox бғҘбғЎбғ”бғҡбғҳбғЎ бғӣбғқбғңбғҳбғўбғқбғ бғҳбғңбғ’бғҳ бғ“бғҗ бғҗбғҰбғ“бғ’бғ”бғңбғҳбғЎ бғЎбғҷбғ бғҳбғһбғўбғҳ (Auto-Reboot)

бғўбғ бғҳбғ’бғ”бғ бғҳ: бғўбғ”бғ бғӣбғҳбғңбғҗбғҡбғҳ бғ’бғҗбғ“бғҳбғЎ бғқбғӨбғҡбғҗбғҳбғңбғЁбғҳ вһ” Make/n8n бғҳбғҰбғ”бғ‘бғЎ бғӣбғ§бғҳбғЎбғҳбғ”бғ  Alert-бғЎ бғҘбғЎбғ”бғҡбғҳбғЎ бғӣбғқбғңбғҳбғўбғқбғ бғҳбғңбғ’бғҳбғЎ бғЎбғҳбғЎбғўбғ”бғӣбғҳбғ“бғҗбғң.

AI/бғҡбғқбғ’бғҳбғҷбғҗ: n8n HTTP API-бғҳбғ— бғЈбғ’бғ–бғҗбғ•бғңбғҳбғЎ бғ’бғҗбғ“бғҗбғўбғ•бғҳбғ бғ—бғ•бғҳбғЎ (Reboot) бғ‘бғ бғ«бғҗбғңбғ”бғ‘бғҗбғЎ бғЁбғ”бғЎбғҗбғ‘бғҗбғӣбғҳбғЎ GPRS/LTE бғ бғқбғЈбғўбғ”бғ бғЎ. бғ”бғҡбғқбғ“бғ”бғ‘бғҗ 5 бғ¬бғЈбғ—бғЎ бғ“бғҗ бғҷбғ•бғҡбғҗбғ• бғҗбғӣбғқбғ¬бғӣбғ”бғ‘бғЎ бғһбғҳбғңбғ’бғЎ.

бғЁбғ”бғ“бғ”бғ’бғҳ: бғ—бғЈ бғҷбғҗбғ•бғЁбғҳбғ бғҳ бғҗбғҰбғ“бғ’бғҗ, бғўбғҳбғҷбғ”бғўбғҳ бғҳбғ®бғЈбғ бғ”бғ‘бғҗ; бғ—бғЈ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗ бғ“бғҗбғ бғ©бғҗ, бғҗбғ•бғўбғқбғӣбғҗбғўбғЈбғ бғҗбғ“ бғҳбғ®бғЎбғңбғ”бғ‘бғҗ бғ‘бғҗбғ бғҗбғ—бғҳ Jira Service Desk-бғЁбғҳ бғ“бғҗ бғ’бғҗбғ“бғҗбғ”бғӘбғ”бғӣбғҗ бғўбғ”бғҘбғңбғҳбғҷбғқбғЎбғЎ.

4. бғӣбғқбғӣбғ®бғӣбғҗбғ бғ”бғ‘бғ”бғҡбғ—бғҗ Viber/Telegram бғ©бғҗбғўбғ‘бғқбғўбғҳ CRM бғЎбғҳбғңбғҘбғ бғқбғңбғҳбғ–бғҗбғӘбғҳбғҳбғ—

бғўбғ бғҳбғ’бғ”бғ бғҳ: бғҷбғҡбғҳбғ”бғңбғўбғҳ Viber/Telegram бғ‘бғқбғўбғҳбғ“бғҗбғң бғҗбғ’бғ–бғҗбғ•бғңбғҳбғЎ бғЁбғ”бғўбғ§бғқбғ‘бғҳбғңбғ”бғ‘бғҗбғЎ бғ©бғҗбғ бғҳбғӘбғ®бғ•бғҳбғЎ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗбғ–бғ”.

AI/бғҡбғқбғ’бғҳбғҷбғҗ: Claude API бғҳбғҰбғ”бғ‘бғЎ бғЁбғ”бғўбғ§бғқбғ‘бғҳбғңбғ”бғ‘бғҗбғЎ, бғҗбғӣбғқбғҳбғӘбғңбғқбғ‘бғЎ бғўбғ”бғ бғӣбғҳбғңбғҗбғҡбғҳбғЎ ID-бғЎ бғ“бғҗ бғһбғ бғқбғ‘бғҡбғ”бғӣбғҗбғЎ, бғҗбғӣбғқбғ¬бғӣбғ”бғ‘бғЎ бғҷбғҡбғҳбғ”бғңбғўбғҳбғЎ бғ‘бғҗбғҡбғҗбғңбғЎбғЎ бғ“бғҗ бғҗбғӣбғ–бғҗбғ“бғ”бғ‘бғЎ бғһбғҗбғЎбғЈбғ®бғЎ.

бғЁбғ”бғ“бғ”бғ’бғҳ: бғҷбғҡбғҳбғ”бғңбғўбғҳ бғӣбғ§бғҳбғЎбғ•бғ” бғҳбғҰбғ”бғ‘бғЎ бғ–бғЈбғЎбғў бғһбғҗбғЎбғЈбғ®бғЎ, бғ®бғқбғҡбғқ бғҳбғңбғӘбғҳбғ“бғ”бғңбғўбғҳ бғ“бғҗ бғӣбғҳбғЎбғҳ AI-бғҷбғҡбғҗбғЎбғҳбғӨбғҳбғҷбғҗбғӘбғҳбғҗ бғҗбғ•бғўбғқбғӣбғҗбғўбғЈбғ бғҗбғ“ бғҳбғ¬бғ”бғ бғ”бғ‘бғҗ OPPA-бғЎ бғЁбғҳбғ“бғҗ CRM-бғЁбғҳ.

5. бғ”бғ бғқбғ•бғңбғЈбғҡбғҳ бғ‘бғҗбғңбғҷбғҳбғЎ (NBG) Compliance бғҗбғңбғ’бғҗбғ бғҳбғЁбғ”бғ‘бғҳбғЎ бғ’бғ”бғңбғ”бғ бғҗбғўбғқбғ бғҳ

бғўбғ бғҳбғ’бғ”бғ бғҳ: Schedule (бғ§бғқбғ•бғ”бғҡбғ—бғ•бғҳбғЈбғ бғҗбғ“, бғ—бғ•бғҳбғЎ 1 бғ бғҳбғӘбғ®бғ•бғЁбғҳ) вһ” n8n бғҡбғқбғҷбғҗбғҡбғЈбғ бғҳ бғЎбғ”бғ бғ•бғ”бғ бғҳбғ“бғҗбғң бғҳбғ¬бғ§бғ”бғ‘бғЎ бғӣбғЈбғЁбғҗбғқбғ‘бғҗбғЎ.

AI/бғҡбғқбғ’бғҳбғҷбғҗ: бғҗбғ’бғ бғқбғ•бғ”бғ‘бғЎ бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғЎ PEP (бғһбғқбғҡбғҳбғўбғҳбғҷбғЈбғ бғҗбғ“ бғҗбғҘбғўбғҳбғЈбғ бғҳ) бғ“бғҗ бғӣбғҗбғҰбғҗбғҡбғҳ бғ бғҳбғЎбғҷбғҳбғЎ бғӣбғ”бғ бғ©бғҗбғңбғўбғ”бғ‘бғ–бғ”, бғҗбғӨбғҳбғҡбғўбғ бғҗбғ•бғЎ PII бғӣбғқбғңбғҗбғӘбғ”бғӣбғ”бғ‘бғЎ бғҡбғқбғҷбғҗбғҡбғЈбғ бғҗбғ“ бғ“бғҗ бғҗбғ¬бғ§бғқбғ‘бғЎ NBG-бғҳбғЎ бғЁбғҗбғ‘бғҡбғқбғңбғҳбғЎ бғӣбғҳбғ®бғ”бғ“бғ•бғҳбғ—.

бғЁбғ”бғ“бғ”бғ’бғҳ: бғҳбғҘбғӣбғңбғ”бғ‘бғҗ бғ“бғҗбғЁбғҳбғӨбғ бғЈбғҡбғҳ бғҗбғңбғ’бғҗбғ бғҳбғЁбғҳ, бғ бғқбғӣбғ”бғҡбғҳбғӘ бғ”бғ’бғ–бғҗбғ•бғңбғ”бғ‘бғҗ Compliance бғқбғӨбғҳбғӘбғ”бғ бғЎ бғ“бғҗбғЎбғҗбғӣбғўбғҷбғҳбғӘбғ”бғ‘бғҡбғҗбғ“, бғ бғҗбғӘ 3 бғ“бғҰбғҳбғҗбғң бғЎбғҗбғӣбғЈбғЁбғҗбғқбғЎ 10 бғ¬бғЈбғ—бғҗбғӣбғ“бғ” бғҗбғӣбғӘбғҳбғ бғ”бғ‘бғЎ.

рҹ§© бғ’бғҗбғӣбғҗбғ бғўбғҳбғ•бғ”бғ‘бғЈбғҡбғҳ No-Code бғ’бғ–бғҗ: Make-бғҳбғЎ бғңбғҗбғўбғҳбғ•бғҳ AI бғӣбғқбғ“бғЈбғҡбғ”бғ‘бғҳ (Anthropic / OpenAI)

бғҗбғ бғҳбғЎ бғ—бғЈ бғҗбғ бғҗ бғ”бғЎ бғЎбғ”бғЎбғҳбғҗ бғ–бғ”бғ“бғӣбғ”бғўбғҗбғ“ бғ бғ—бғЈбғҡбғҳ бғҗбғ бғҗбғўбғ”бғҘбғңбғҳбғҷбғЈбғ бғҳ бғ—бғҗбғңбғҗбғӣбғЁбғ бғқбғӣбғҡбғ”бғ‘бғҳбғЎбғ—бғ•бғҳбғЎ?
бғңбғ”бғ“бғҡбғҳ HTTP API бғӣбғқбғ—бғ®бғқбғ•бғңбғ”бғ‘бғҳбғЎбғҗ бғ“бғҗ JSON бғЎбғҘбғ”бғӣбғ”бғ‘бғҳбғЎ бғ¬бғ”бғ бғҗ бғ«бғҳбғ бғҳбғ—бғҗбғ“бғҗбғ“ бғ“бғ”бғ•бғ”бғҡбғқбғһбғ”бғ бғ”бғ‘бғҳбғЎбғ—бғ•бғҳбғЎбғҗбғҗ. бғ‘бғҳбғ–бғңбғ”бғЎ-бғӣбғ”бғңбғ”бғҜбғ”бғ бғ”бғ‘бғЎбғҗ бғ“бғҗ бғҗбғңбғҗбғҡбғҳбғўбғҳбғҷбғқбғЎбғ”бғ‘бғЎ бғЁбғ”бғЈбғ«бғҡбғҳбғҗбғ— бғ’бғҗбғӣбғқбғҳбғ§бғ”бғңбғқбғң Make-бғҳбғЎ бғңбғҗбғўбғҳбғ•бғҳ (Native) AI бғӣбғқбғ“бғЈбғҡбғ”бғ‘бғҳ, бғ бғҗбғӘ бғЎбғ бғЈбғҡбғҗбғ“ бғҗбғҘбғ бғқбғ‘бғЎ бғҷбғқбғ“бғҳбғЎ бғ¬бғ”бғ бғҳбғЎ бғЎбғҗбғӯбғҳбғ бғқбғ”бғ‘бғҗбғЎ:

рҹӣ пёҸ бғӣбғ”бғ—бғқбғ“бғҳ бғҗ: бғңбғҗбғўбғҳбғ•бғҳ Anthropic / OpenAI бғӣбғқбғ“бғЈбғҡбғҳ

  • бғЈбғ‘бғ бғҗбғҡбғқбғ“ бғҗбғҳбғ бғ©бғ”бғ•бғ— бғӣбғқбғ“бғЈбғҡбғЎ: Anthropic вҶ’ Create a Message.
  • API бғ’бғҗбғЎбғҗбғҰбғ”бғ‘бғЎ бғ©бғҗбғ¬бғ”бғ бғ— бғӣбғҗбғ бғўбғҳбғ• бғ•бғ”бғҡбғЁбғҳ (dropdown).
  • бғӣбғқбғ“бғ”бғҡбғЎ бғҗбғҳбғ бғ©бғ”бғ•бғ— бғЎбғҳбғҳбғ“бғҗбғң (claude-3-5-sonnet).
  • бғһбғ бғқбғӣбғһбғўбғЎ бғ¬бғ”бғ бғ— бғ©бғ•бғ”бғЈбғҡбғ”бғ‘бғ бғҳбғ• бғўбғ”бғҘбғЎбғўбғҗбғ“ JSON бғӨбғқбғ бғӣбғҗбғўбғҳбғЎ бғ’бғҗбғ бғ”бғЁбғ”.
  • вң“ бғҳбғ“бғ”бғҗбғҡбғЈбғ бғҳбғҗ бғЎбғ¬бғ бғҗбғӨбғҳ бғ“бғҗ бғӣбғҗбғ бғўбғҳбғ•бғҳ бғҗбғӣбғқбғӘбғҗбғңбғ”бғ‘бғҳбғЎбғ—бғ•бғҳбғЎ.

рҹҢҗ бғӣбғ”бғ—бғқбғ“бғҳ бғ‘: HTTP бғӣбғқбғ“бғЈбғҡбғҳ (бғ“бғ”бғ•бғ”бғҡбғқбғһбғ”бғ бғҳбғЎ бғ’бғ–бғҗ)

  • бғ¬бғ”бғ бғ— бғңбғ”бғ“бғҡ POST бғӣбғқбғ—бғ®бғқбғ•бғңбғҗбғЎ Anthropic-бғҳбғЎ API бғҡбғҳбғңбғҷбғ–бғ”.

  • вҶ’Ignore: бғЈбғ’бғЈбғҡбғ”бғ‘бғ”бғҡбғ§бғқбғӨбғЎ бғ®бғҗбғ бғ•бғ”бғ–бғЎ бғ“бғҗ бғҗбғ’бғ бғ«бғ”бғҡбғ”бғ‘бғЎ бғЎбғӘбғ”бғңбғҗбғ бғЎ (бғӣбғҗбғ’. бғ—бғЈ Slack бғ“бғҗбғ”бғӘбғҗ, бғўбғҳбғҷбғ”бғўбғҳ бғӣбғҗбғҳбғңбғӘ бғҳбғ¬бғ”бғ бғ”бғ‘бғҗ бғ‘бғҗбғ–бғҗбғЁбғҳ).
