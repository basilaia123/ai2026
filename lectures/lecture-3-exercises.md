## ✍️ ვარჯიში 4: რედაქტირება და გაუმჯობესება (25 წუთი)

### ამოცანა 4.1: Verbose → Concise

**ორიგინალი ტექსტი:**
```
"ჩვენი კომპანია, რომელიც დაარსდა 2020 წელს, არის ერთ-ერთი ყველაზე წამყვანი და ინოვაციური ორგანიზაცია software development-ის ინდუსტრიაში. ჩვენ ვთავაზობთ ფართო სპექტრის სერვისებს და პროდუქტებს, რომლებიც დიზაინებულია იმისთვის რომ დაეხმაროს ბიზნესებს მათი ციფრული ტრანსფორმაციის მიღწევაში და უკეთესი შედეგების მიღებაში."
```

**თქვენი ამოცანა:**
```
შემოკლე ეს ტექსტი 50%-ით, შეინარჩუნე მთავარი მესიჯი.

მოთხოვნები:
- Remove redundancy
- Keep key facts
- Tighter sentences
- Maintain professional tone
```

### ამოცანა 4.2: Passive → Active Voice

**ორიგინალი:**
```
"The project was completed by the team ahead of schedule. Multiple challenges were encountered during the implementation phase, but solutions were found by the developers. The final product has been tested and approved by the client."
```

**თქვენი ამოცანა:**
```
გადაწერე active voice-ით.

მოთხოვნები:
- All sentences active
- More dynamic
- Keep information
- Professional tone
```

### ამოცანა 4.3: Jargon → Simple Language

**ორიგინალი:**
```
"Our solution leverages cutting-edge AI algorithms to synergize cross-functional workflows, enabling seamless operationalization of data-driven paradigm shifts that optimize KPIs and drive holistic value creation across the enterprise ecosystem."
```

**თქვენი ამოცანა:**
```
გამარტივე non-technical აუდიტორიისთვის.

მოთხოვნები:
- Remove buzzwords
- Plain English
- Concrete examples
- Still professional
```

## 📝 ვარჯიში 5: Copywriting ფორმულები (30 წუთი)

### მიზანი
სხვადასხვა copywriting formula-ს გამოცდა

### ამოცანა

შექმენით რეკლამური ტექსტი "Online Fitness App"-ისთვის გამოყენებით 3 განსხვავებული ფორმულა.

**პროდუქტი:**
- სახელი: FitFlow
- ფასი: 15 ლარი/თვე
- ფიჩები: personalized workouts, nutrition plans, progress tracking
- სამიზნე: busy professionals 25-40 წლის

#### 5.1 AIDA Formula
```
დაწერე ad copy გამოყენებით AIDA (Attention, Interest, Desire, Action):

Attention: [Hook headline]
Interest: [Why they should care]
Desire: [Benefits და emotional appeal]
Action: [Clear CTA]

სიგრძე: 100-125 სიტყვა
ფორმატი: landing page hero section
```

#### 5.2 PAS Formula
```
დაწერე ad copy გამოყენებით PAS (Problem, Agitate, Solution):

Problem: [Pain point]
Agitate: [Make them feel it]
Solution: [How product solves it]

სიგრძე: 100-125 სიტყვა
ფორმატი: email campaign
```

#### 5.3 BAB Formula
```
დაწერე ad copy გამოყენებით BAB (Before, After, Bridge):

Before: [Current struggle]
After: [Desired outcome]
Bridge: [How to get there]

სიგრძე: 100-125 სიტყვა
ფორმატი: social media ad
```

## 🔍 ვარჯიში 8: SEO-Optimized Blog Post (35 წუთი)

### მიზანი
SEO-optimized კონტენტის შექმნა

**თემა:** "How to Start a Blog in 2025"

**SEO პარამეტრები:**
- **Primary Keyword:** "how to start a blog"
- **Secondary Keywords:** "blogging for beginners", "create a blog", "blog setup"
- **Search Intent:** Informational (step-by-step guide)
- **Target Length:** 1200-1500 სიტყვა

#### 8.1 სტრუქტურის შექმნა
```
შექმენი blog post სტრუქტურა:

Introduction (150 სიტყვა):
  - Hook
  - Why start a blog in 2025?
```
