ლექცია 5: ინფორმაციის მოძიება და გამოყენება

პრაქტიკული სავარჯიშოები - AI პრაქტიკული კურსი

📚 სავარჯიშოების მიმოხილვა

ეს სავარჯიშოები დაგეხმარებათ, რომ გამოიყენოთ AI ინფორმაციის სწრაფი მოძიებისთვის, დამუშავებისა და პრაქტიკულ სამუშაოში გამოყენებისთვის. თითოეული სავარჯიშო ორიენტირებულია კონკრეტულ ტექნიკასა და რეალურ სცენარებზე.

საერთო რჩევები:

  • გამოიყენეთ ChatGPT, Claude, Perplexity.ai ან Gemini
  • ყოველთვის შეამოწმეთ ფაქტები
  • შექმენით პრომპტების ბიბლიოთეკა
  • შეინახეთ სასარგებლო შედეგები
  • ფოკუსირდით პრაქტიკულ გამოყენებაზე

სავარჯიშოების სირთულე:

  • 🟢 ადვილი (10-15 წუთი)
  • 🟡 საშუალო (20-30 წუთი)
  • 🔴 რთული (45-60 წუთი)

სავარჯიშო 1: სწრაფი პასუხები სამუშაო კითხვებზე 🟢 ადვილი

მიზანი:

ისწავლეთ AI-ის გამოყენება სწრაფი, კონკრეტული პასუხების მისაღებად

დავალება:

აირჩიეთ 5 ტიპური კითხვა თქვენი სამუშაოდან და მოიძიეთ პასუხები AI-ის დახმარებით:

პრომპტის შაბლონი: "მჭირდება სწრაფი, კონკრეტული პასუხი: [თქვენი კითხვა] გთხოვთ: - პასუხი იყოს 2-3 წინადადება - იყოს კონკრეტული და პრაქტიკული - ფოკუსირება გააკეთეთ მთავარ პუნქტებზე"

კითხვების მაგალითები:

  • "როგორ დავიწყო პროფესიონალური საქმიანი წერილი?"
  • "რა განსხვავებაა JSON და XML ფორმატებს შორის?"
  • "როგორ ვუთხრა კლიენტს ვადის გაგრძელების შესახებ?"

რას უნდა მიიღოთ:

  • 5 კითხვა და 5 სწრაფი პასუხი
  • თითოეული პასუხი 50-100 სიტყვის ფარგლებში
  • დოკუმენტირებული პრომპტები

თვითშეფასება:

  • პასუხები კონკრეტული და გამოსადეგია
  • ინფორმაცია მარტივად არის გადმოცემული
  • პრომპტები ხელახლა გამოყენებადია

დრო: 10-15 წუთი

სავარჯიშო 2: ტექსტის 3 ტიპის შეჯამება 🟢 ადვილი

მიზანი:

ისწავლეთ ერთი ტექსტის სხვადასხვა ფორმატში შეჯამება

დავალება:

აირჩიეთ გრძელი სტატია ან დოკუმენტი (800+ სიტყვა) და შექმენით 3 სახის შეჯამება:

1. Executive Summary (50-75 სიტყვა):

შეაჯამეთ ქვემოთ მოცემული ტექსტი 50-75 სიტყვაში. ფოკუსირება გააკეთეთ მთავარ გადაწყვეტილებებსა და რეკომენდაციებზე. გამოიყენეთ ბიზნესლიდერებისთვის გასაგები ენა: [ტექსტი]

2. Key Points (5 ძირითადი პუნქტი):

ჩამოწერეთ 5 მთავარი პუნქტი ამ ტექსტიდან. თითოეული პუნქტი უნდა იყოს: - ერთი მოკლე წინადადება - კონკრეტული ფაქტი - ლოგიკური თანმიმდევრობით [ტექსტი]

3. მარტივი ახსნა (100 სიტყვა):

ახსენით ეს ტექსტი ისე, რომ 12 წლის ბავშვისთვისაც გასაგები იყოს. გამოიყენეთ მარტივი სიტყვები და ყოველდღიური მაგალითები. [ტექსტი]

რას უნდა მიიღოთ:

  • ორიგინალი ტექსტი
  • 3 განსხვავებული შეჯამება
  • ანალიზი: რომელი ფორმატი რა სიტუაციაშია უფრო გამოსადეგი

თვითშეფასება:

  • სამივე შეჯამება ერთმანეთისგან განსხვავდება
  • შენარჩუნებულია მთავარი სათქმელი
  • თითოეული ფორმატი სხვადასხვა აუდიტორიისთვისაა გამოსადეგი

დრო: 15-20 წუთი

სავარჯიშო 3: ინფორმაციის შედარება და გადაწყვეტილების მიღება 🟡 საშუალო

მიზანი:

ისწავლეთ 3 ვარიანტის შედარებითი ანალიზი გადაწყვეტილების მისაღებად

დავალება:

აირჩიეთ რეალური გადაწყვეტილება თქვენი სამუშაოდან და შეადარეთ 3 ვარიანტი:

ნაბიჯი 1: ინფორმაციის მოძიება

მჭირდება ინფორმაცია [თემა]-ის შესახებ: გთხოვთ, შეადაროთ შემდეგი 3 ვარიანტი: 1. [ვარიანტი 1] 2. [ვარიანტი 2] 3. [ვარიანტი 3] თითოეულისთვის მომაწოდეთ: - ძირითადი მახასიათებლები - ფასი/ღირებულება - დადებითი მხარეები - უარყოფითი მხარეები

ნაბიჯი 2: შედარების ცხრილის შექმნა

შექმენით ცხრილი ამ სტრუქტურით:

  • კრიტერიუმი 1: ფასი
  • კრიტერიუმი 2: ფუნქციები
  • კრიტერიუმი 3: გამოყენების სიმარტივე
  • კრიტერიუმი 4: მხარდაჭერა
  • კრიტერიუმი 5: მომხმარებელთა შეფასებები

ნაბიჯი 3: ფაქტების შემოწმება

შეამოწმეთ შემდეგი ინფორმაცია: [AI-ის მიერ მოწოდებული ფაქტი] მოიძიეთ 2-3 დამოუკიდებელი წყარო და დაადასტურეთ ან უარყავით.

ნაბიჯი 4: რეკომენდაცია

დაწერეთ 150-200 სიტყვიანი რეკომენდაცია, თუ რომელი ვარიანტია საუკეთესო და რატომ.

რას უნდა მიიღოთ:

  • კვლევის დოკუმენტი 3 ვარიანტით
  • შედარების ცხრილი (3×5)
  • ფაქტების გადამოწმების შედეგები
  • დასაბუთებული რეკომენდაცია

თვითშეფასება:

  • შედარება ობიექტურია
  • ფაქტები შემოწმებულია
  • რეკომენდაცია დასაბუთებულია
  • შედეგი პრაქტიკულად გამოსაყენებადია

დრო: 25-30 წუთი

სავარჯიშო 4: მარტივი ანგარიშის შექმნა 🟡 საშუალო

მიზანი:

ისწავლეთ AI-ის დახმარებით ინფორმაციის სწრაფად შეგროვება და მოკლე ანგარიშის შექმნა

დავალება:

შექმენით 1-გვერდიანი ანგარიში თქვენი სფეროდან რომელიმე თემაზე:

ეტაპი 1: ინფორმაციის შეგროვება

მჭირდება ინფორმაცია [თემა]-ის შესახებ 1-გვერდიანი ანგარიშისთვის: 1. რა არის [თემა]? (50-75 სიტყვა) 2. მიმდინარე სიტუაცია/სტატუსი (75-100 სიტყვა) 3. 3 მთავარი ტენდენცია 4. 5 ძირითადი დასკვნა 5. 3 რეკომენდაცია მომავლისთვის

ეტაპი 2: სტრუქტურის შექმნა

ანგარიში უნდა შეიცავდეს:

  • სათაური და თარიღი
  • მოკლე შეჯამება (Executive Summary) (3-4 წინადადება)
  • მიმდინარე სიტუაცია (1 პარაგრაფი)
  • ძირითადი დასკვნები (5 პუნქტი)
  • რეკომენდაციები (3 პუნქტი)

ეტაპი 3: ვიზუალიზაცია

შექმენით მარტივი ცხრილი ან სია, რომელიც აჩვენებს: [კონკრეტული მონაცემები, რომლის ვიზუალიზაციაც გჭირდებათ] ფორმატი: markdown table ან bullets

რას უნდა მიიღოთ:

  • 1-გვერდიანი ანგარიში (400-500 სიტყვა)
  • სტრუქტურირებული და მარტივად წასაკითხი
  • მინიმუმ 1 ცხრილი ან ორგანიზებული სია
  • წარსადგენად მზა დოკუმენტი

თვითშეფასება:

  • ანგარიში ნათლად არის სტრუქტურირებული
  • ინფორმაცია რელევანტურია
  • მოკლე შეჯამება ლაკონური და ინფორმაციულია
  • ვიზუალური ელემენტები აუმჯობესებს ընկալումას

დრო: 25-30 წუთი

სავარჯიშო 5: ფაქტების სწრაფი შემოწმება 🟢 ადვილი

მიზანი:

ისწავლეთ AI-ის დახმარებით ინფორმაციის სიზუსტის სწრაფად შემოწმება

დავალება:

შეამოწმეთ 5 ფაქტი/ინფორმაცია, რომელიც გსმენიათ ან წაგიკითხავთ:

პრომპტი თითოეული ფაქტისთვის:

შეამოწმეთ შემდეგი ინფორმაცია: "[ფაქტი]" გთხოვთ: 1. დაადასტურეთ ან უარყავით ეს ინფორმაცია 2. მიუთითეთ, რამდენად ზუსტია (სრულად ზუსტი / ნაწილობრივ ზუსტი / არასწორი) 3. მომაწოდეთ 2-3 სანდო წყარო 4. თუ არასწორია, მიუთითეთ სწორი ინფორმაცია

მაგალითი:

ფაქტი: "საქართველოში პირველი AI კომპანია 2015 წელს დაარსდა"

შედეგი: დადასტურება/უარყოფა + წყაროები + კონტექსტი

რას უნდა მიიღოთ:

  • 5 ფაქტის შემოწმება
  • თითოეულისთვის: სტატუსი + წყაროები
  • სწორი ინფორმაცია, თუ ფაქტი არასწორი იყო
⚠️ მნიშვნელოვანი: ყოველთვის გადაამოწმეთ AI-ის პასუხებიც! სასურველია Perplexity.ai-ის გამოყენება, რადგან ის ავტომატურად უთითებს წყაროებს.

თვითშეფასება:

  • ხუთივე ფაქტი შემოწმებულია
  • წყაროები მითითებულია
  • გაქვთ ფაქტების სწრაფად გადამოწმების მეთოდი

დრო: 12-15 წუთი

სავარჯიშო 6: პირადი პრომპტების ბიბლიოთეკა 🟡 საშუალო

მიზანი:

შექმენით პერსონალური, მზა პრომპტების კოლექცია

დავალება:

შექმენით 10 პრომპტის შაბლონი თქვენი ყოველდღიური სამუშაოსთვის:

კატეგორიები (თითოეულში 2 პრომპტი):

1. სწრაფი პასუხები:

შაბლონი: "მჭირდება კონკრეტული პასუხი [თემა]-ზე: [კითხვა]. პასუხი უნდა იყოს 2-3 წინადადება, პრაქტიკული და გასაგები."

2. ტექსტის შეჯამება:

შაბლონი: "შეაჯამეთ შემდეგი ტექსტი [ფორმატი] ფორმატში, [რაოდენობა] სიტყვით. ფოკუსირება გააკეთეთ [თემა/ასპექტი]-ზე: [ტექსტი]"

3. რთული თემის ახსნა:

შაბლონი: "ახსენით [თემა] მარტივად, [აუდიტორია]-სთვის გასაგები ენით. გამოიყენეთ ყოველდღიური მაგალითები."

4. ინფორმაციის შედარება:

შაბლონი: "შეადარეთ [A] და [B] ცხრილის სახით. კრიტერიუმები: [კრიტერიუმი1], [კრიტერიუმი2], [კრიტერიუმი3]."

5. სწრაფი კვლევა:

შაბლონი: "მოიძიეთ [რაოდენობა] საუკეთესო [რესურსი/ხელსაწყო/მეთოდი] [მიზანი]-სთვის. თითოეულისთვის მიუთითეთ: სახელი, მოკლე აღწერა, სარგებელი."

რას უნდა მიიღოთ:

სავარჯიშო 8: ყოვლისმომცველი საინფორმაციო პროექტი 🔴 რთული
