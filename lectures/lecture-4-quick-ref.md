## 🧠 ბრეინსტორმინგის ტექნიკები

### 1. SCAMPER

```
S - Substitute (შენაცვლება)
C - Combine (გაერთიანება)
A - Adapt (ადაპტირება)
M - Modify (მოდიფიკაცია)
P - Put to other use (სხვა გამოყენება)
E - Eliminate (ამოღება)
R - Reverse (შებრუნება)
```

**Prompt:**
```
გამოიყენე SCAMPER [პროდუქტი]-სთვის.
თითოეული კატეგორიისთვის 3 იდეა.
```

### 2. What If სცენარები

```
დააგენერირე 10 "What If" სცენარი [თემა]-სთვის.
თითოეულისთვის: შედეგი + რეკომენდაცია
```

### 3. Reverse Brainstorming

```
როგორ გავაუარესო [მიზანი]?
→ 15 გზა გაუარესებისთვის
→ შემდეგ შებრუნება: როგორ ავიცილოთ თავიდან
```

### 4. Random Input

```
1. დააგენერირე 5 შემთხვევითი სიტყვა
2. დააკავშირე [თქვენს პროექტს]
3. თითოეულისთვის 3 იდეა
```

### 5. Mind Mapping

```
შექმენი mind map:
- ცენტრალური იდეა
- 5-7 მთავარი შტო
- თითოზე 3-5 ქვეშტო
- კავშირები
```

## ⚖️ იდეების შეფასება

### სწრაფი Matrix

შეაფასე 1-10 სკალით:

| კრიტერიუმი | აღწერა |
|-----------|--------|
| **Impact** | რა ეფექტი ექნება? |
| **Feasibility** | რამდენად ადვილია? |
| **Cost** | რამდენი დაჯდება? |
| **Time** | რა დრო დასჭირდება? |
| **Innovation** | რამდენად ახალია? |
| **Market Fit** | საჭიროებას პასუხობს? |

### Prioritization Quadrants

- **Quick Wins:** High Impact + Low Effort
- **Strategic:** High Impact + High Effort
- **Fill-ins:** Low Impact + Low Effort
- **Time Sinks:** Low Impact + High Effort

## 🎯 5-წუთიანი Ideation Workflow

1. **აირჩიე** ტექნიკა (SCAMPER, What If, Random...)
2. **დააგენერირე** 10+ იდეა (არ გააფილტრო!)
3. **შეაფასე** matrix-ით
4. **აირჩიე** top 3
5. **დაახვეწე** 1 იდეა
6. **პროტოტიპირება** → test
