📊 სავარჯიშო 3: Create a Presentation with Gamma

ხანგრძლივობა: 20 წუთი | რთულება: ⭐⭐

📝 დავალება:

შექმენით 8-10 სლაიდიანი პრეზენტაცია Gamma.app-ის გამოყენებით შემდეგ თემაზე (აირჩიეთ ერთი):

🎯 თემის ვარიანტები (აირჩიეთ 1):

  1. Remote Work Best Practices 2024
  2. Social Media Marketing Strategy
  3. Introduction to Cybersecurity
  4. Healthy Lifestyle Tips
  5. თქვენი პროფესიის თემა

📋 პრეზენტაციამ უნდა მოიცავდეს:

  1. Title slide
  2. Introduction / Problem Statement
  3. 3-4 Main Points (თითოეული ცალკე სლაიდზე)
  4. Examples ან Case Studies
  5. Key Takeaways
  6. Conclusion / Call to Action

✍️ Gamma Prompt მაგალითი:

"Create a 10-slide presentation about Remote Work Best Practices in 2024. Include:
- Title slide with engaging visual
- Introduction: Why remote work matters
- Current statistics
- Top 5 best practices (communication, time management, work-life balance, tech tools, team building)
- Common challenges and solutions
- Tools and technologies
- Case study example
- Implementation roadmap
- Conclusion with action steps
Use professional design with modern visuals."

🎯 რა უნდა მოახდინოთ:

  • გაუშვით Gamma-ს AI-ს სრული პრეზენტაციის შესაქმნელად
  • გადახედეთ და დაარედაქტირეთ რამდენიმე სლაიდი
  • შეარჩიეთ თემა/დიზაინი რომელიც მოგწონთ
  • დაამატეთ ან შეცვალეთ სურათები
  • გაზიარეთ link-ით ან export PDF

🎭 სავარჯიშო 5: Social Media Visual Campaign

ხანგრძლივობა: 30 წუთი | რთულება: ⭐⭐⭐⭐

📝 დავალება:

შექმენით mini visual campaign სოციალური მედიისთვის. წარმოიდგინეთ რომ ამზადებთ პროდუქტის ან სერვისის გაშვების კამპანიას.

🎯 რა უნდა შექმნათ:

  1. Instagram Feed Post (1080x1080):
    • Hero სურათი პროდუქტით/სერვისით
    • პროფესიონალური და ელეგანტური
  2. Instagram Story (1080x1920):
    • Teaser ან announcement
    • ტექსტი overlay-ით
  3. LinkedIn Post Visual (1200x627):
    • უფრო ფორმალური, ბიზნეს-ორიენტირებული
    • ფოკუსი value proposition-ზე

🎨 Brand Consistency რჩევები:

  • გამოიყენეთ იგივე ფერთა პალიტრა ყველა სურათში
  • თანმიმდევრული ფონტები
  • მსგავსი სტილი და განათება
  • ერთიანი ვიზუალური ენა

📝 Workflow:

  1. გადაწყვიტეთ პროდუქტი/სერვისი (შეიძლება ფიქტიური)
  2. შექმენით ფერთა პალიტრა (2-3 ძირითადი ფერი)
  3. დაწერეთ prompts სურათების გენერირებისთვის
  4. გენერირეთ base სურათები AI-ით
  5. დაარედაქტირეთ Canva-ში:
    • შეცვალეთ ზომები თითოეული პლატფორმისთვის
    • დაამატეთ ტექსტი
    • დაამატეთ ბრენდის ელემენტები

🎯 შენახვა:

  • 3 სურათი სწორი ზომებით
  • ყველა prompt-ის დოკუმენტაცია
  • რომელი ინსტრუმენტები გამოიყენეთ
  • მოკლე აღწერა (2-3 წინადადება) თითოეული სურათის კონცეფციისა
