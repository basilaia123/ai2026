  4. მიუწერეთ რა დარჩეს უცვლელი და რა შეიცვალოს ახალ გარემოში.

Prompt 1: Create a confident Georgian businesswoman in her early 30s, beige blazer, dark brown hair, minimal makeup, premium corporate portrait, clean soft lighting, realistic photography.

Prompt 2: Use the first generated image as the character reference. Keep the same face, hairstyle, age, outfit identity, and premium realistic style. Place her in a modern conference room giving a presentation. Change only the setting and camera angle.

🖼️ ChatGPT Images

ორი ნაბიჯი: generate first, then reuse the same person

  1. Prompt 1: დააგენერირეთ საწყისი პერსონა.

  4. შეცვალეთ მხოლოდ setting, pose, angle ან background.
