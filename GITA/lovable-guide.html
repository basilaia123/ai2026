რა არის Lovable?

Lovable (ყოფილი GPT Engineer) არის AI-ზე დაფუძნებული პლატფორმა, რომელიც საშუალებას გაძლევთ, შექმნათ სრულფასოვანი ვებ აპლიკაციები მხოლოდ ტექსტური აღწერილობის საშუალებით. თქვენ აღწერთ, რა გსურთ, ხოლო Lovable წერს კოდს, ქმნის დიზაინს და უზრუნველყოფს აპლიკაციის ინტერნეტში განთავსებას (deployment).

ვისთვის არის Lovable?

  • არაპროგრამისტები: ბიზნესის მფლობელები, მარკეტოლოგები, დიზაინერები
  • სტარტაპები: სწრაფი პროტოტიპირება და MVP-ის შექმნა
  • დეველოპერები: იდეების სწრაფი ტესტირება
  • სტუდენტები: პროექტების შექმნა სწავლის პროცესში

ძირითადი შესაძლებლობები

1. Full-Stack აპლიკაციები

Lovable ქმნის სრულფასოვან აპლიკაციებს:

  • Frontend: React, TypeScript, Tailwind CSS
  • Backend: Supabase (მონაცემთა ბაზა + ავტორიზაცია)
  • Hosting: ავტომატური განთავსება (hosting)

2. რედაქტირება რეალურ დროში

ცვლილებებს რეალურ დროში ხედავთ. თქვენ შეგიძლიათ:

  • ჩათში მისცეთ ახალი ინსტრუქციები
  • პირდაპირ კოდში შეიტანოთ ცვლილებები
  • ვიზუალურად შეცვალოთ ელემენტები

3. ინტეგრაციები

  • Supabase: მონაცემთა ბაზა და ავტორიზაცია
  • Stripe: გადახდების სისტემა
  • GitHub: კოდის ექსპორტი და სინქრონიზაცია
  • სხვა API-ები: ნებისმიერი REST API

შეზღუდვები

  • რთული ბიზნეს-ლოგიკა შეიძლება მოითხოვდეს ხელით კორექტირებას
  • უფასო ვერსიას აქვს გარკვეული შეზღუდვები

დემო პროექტი: დავალებების მენეჯერი (To-Do App)

შევქმნათ მარტივი To-Do აპლიკაცია Lovable-ით ნაბიჯ-ნაბიჯ.

ნაბიჯი 1: პროექტის შექმნა

გადადით lovable.dev-ზე და დარეგისტრირდით (შეგიძლიათ გამოიყენოთ GitHub ანგარიში).

დააჭირეთ ღილაკს "New Project" და შეიყვანეთ პირველი პრომპტი (დავალება):

Create a modern to-do list application with the following features: - Add new tasks with a text input and button - Mark tasks as complete (checkbox) - Delete tasks - Filter tasks: All, Active, Completed - Show task count - Clean, minimal design with purple accent color - Responsive layout for mobile

Lovable გააანალიზებს თქვენს მოთხოვნას და დაიწყებს კოდის გენერაციას. დაელოდეთ 30-60 წამი.

ნაბიჯი 2: შედეგის ნახვა

Lovable შექმნის სამუშაო აპლიკაციას:

  • მარცხენა პანელი: ჩათი ხელოვნურ ინტელექტთან
  • შუა პანელი: კოდის რედაქტორი
  • მარჯვენა პანელი: აპლიკაციის წინასწარი ნახვა (Preview)
[Lovable ინტერფეისის სქემა: ჩათი | კოდი | წინასწარი ნახვა]

ნაბიჯი 3: გაუმჯობესება ჩათის მეშვეობით

მოდით, დავამატოთ ახალი ფუნქციები:

Add these improvements: 1. Add due date picker for each task 2. Add priority levels (Low, Medium, High) with color coding 3. Save tasks to local storage so they persist after refresh 4. Add a dark mode toggle button

Lovable ავტომატურად განაახლებს კოდს და გამოჩნდება ახალი ფუნქციები.

ნაბიჯი 4: დიზაინის კორექტირება

შეგიძლიათ ვიზუალურად შეცვალოთ ელემენტები ან მისცეთ დიზაინის ინსტრუქციები:

Update the design: - Make the header gradient from purple (#667eea) to pink (#764ba2) - Add subtle shadow to task cards - Use rounded corners (12px) for all elements - Add smooth animations when adding/removing tasks

ნაბიჯი 5: გამოქვეყნება

როცა კმაყოფილი იქნებით:

  1. დააჭირეთ "Share" ღილაკს
  2. აირჩიეთ "Publish" (გამოქვეყნება)
  3. მიიღებთ ბმულს: your-app.lovable.app
