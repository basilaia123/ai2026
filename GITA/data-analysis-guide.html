1. რა შეცვალა AI-მ მონაცემთა ანალიზში?

აქამდე მოცულობითი ცხრილების ანალიზი, Pivot Table-ების აწყობა, პითონით პროგნოზირება და ვიზუალიზაცია საჭიროებდა Excel/SQL/Python-ის სიღრმისეულ ცოდნას ან ძვირადღირებულ პროგრამულ უზრუნველყოფას (მაგ. PowerBI, Tableau). დღეს უფროსი მონაცემთა მეცნიერის (Senior Data Scientist) ძალა თქვენს ბრაუზერშია.

რა არის 'Code Interpreter' (Advanced Data Analysis)?

როდესაც ChatGPT-ში ატვირთავთ ექსელს, ის უბრალოდ ტექსტურად კი არ კითხულობს მას. ის კულისებში წერს Python-ის მოკლე სკრიპტს, უშვებს ამ კოდს სანდო გარემოში (Sandbox) და პასუხად გაძლევთ ამ კოდის მიერ დათვლილ, 100%-ით ზუსტ მათემატიკურ პასუხს. ამის გამო, ის აღარ "ჰალუცინაციებს" რიცხვებზე.

2. კონფიდენციალურობა და ფაილების სწორი ატვირთვა

პირველი ნაბიჯია AI-სთვის მონაცემების მიწოდება. ეს შეიძლება ორი გზით მოხდეს: უშუალოდ ცხრილის კოპირება/ჩასმა (პატარა ზომის მონაცემებისთვის) ან ფაილის პირდაპირი ტვირთვა (დიდი .csv ან .xlsx ბაზებისთვის).

⚠️ კონფიდენციალურობის (Privacy) ოქროს წესი

არასოდეს ატვირთოთ PII (Personally Identifiable Information).

ატვირთვამდე Excel-ში ყოველთვის ჩაატარეთ "ანონიმიზაცია". მაგალითად, წაშალეთ კლიენტების სახელები და მობილურის ნომრები და დატოვეთ მხოლოდ `Client_ID`, `Age`, `Purchase_Value`.

აწყობის (Setup) საუკეთესო პრომპტი ატვირთვისას:

ფაილის ატვირთვისთანავე ნუ მოსთხოვთ პირდაპირ რთულ ანალიზს. სჯობს ჯერ მიაწოდოთ კონტექსტი.

მე აგიტვირთე CRM-დან გადმოწერილი B2B გაყიდვების ანონიმიზებული ბაზა. გთხოვთ ჯერ: 1. წაიკითხე სვეტები და მომწერე გესმის თუ არა რას ნიშნავს თითოეული. 2. მითხარი თუ ხედავ რაიმე ტექნიკურ შეცდომას (ცარიელი ველები, არეული თარიღები). 3. დამიდასტურე მზადყოფნა და მხოლოდ ამის შემდეგ მოგცემ კონკრეტულ დავალებებს.

3. არასტრუქტურირებული მონაცემების სტრუქტურირება

ძალიან ხშირად, ინფორმაცია ცხრილში კი არ გვაქვს, არამედ გაფანტულია PDF-ებში, Word დოკუმენტებში, კონტრაქტებსა თუ მეილებში. AI-ის ერთ-ერთი ყველაზე ჯადოსნური უნარი **არასტრუქტურირებული ტექსტის სტრუქტურირებაა**.

სცენარი: PDF ინვოისების ან ხელშეკრულებების ექსელად ქცევა

თუ გაქვთ 50-გვერდიანი PDF კვლევა და გინდათ შიგნიდან გამოკრიფოთ ყველა კომპანიის სახელი, შემოსავალი და წელი, ატვირთეთ PDF და გამოიყენეთ ეს პრომპტი:

ატვირთულია 50-გვერდიანი PDF დოკუმენტი (ფინანსური რეპორტი). წაიკითხე მთლიანი დოკუმენტი და ამოიღე შემდეგი მონაცემები ექსელის (CSV) ფორმატში: - კომპანიის დასახელება - ნახსენები ფინანსური წელი - აღწერილი წმინდა მოგება - მთავარი რისკ ფაქტორი (მოკლე 1 წინადადებით) გამოტოვე ყველა სხვა ტექსტი. თუ რამე ველს ვერ პოულობ, ჩაწერე "N/A". მიპასუხე პირდაპირ გადმოსაწერი CSV ფაილით.

4. Data Cleaning (მონაცემების გასუფთავება) - რუტინის განადგურება

ექსელში მონაცემების დალაგება - მაგალითად სხვადასხვა ენაზე დაწერილი ქალაქების გაერთიანება, სატელეფონო ინდექსების გასწორება - საათობით დროს მოითხოვს. AI-სთან ერთად ეს სრულიად ავტომატურია.

გიგზავნით "Raw_Survey_Data.csv" ფაილს გამოკითხვის შედეგებით. გთხოვთ ჩაატარო "Data Cleaning" პროცესი: 1. წაშალე სტრიქონები, სადაც "Email" ან "Phone" ორივე ცარიელია. 2. "Date_of_Birth" სვეტში თარიღები არის სამი სხვადასხვა ფორმატითაა (12/05/2020, 2020-05-12, მაისი 12, 2020). ყველა გადაიყვანე ერთიან ISO ფორმატით (YYYY-MM-DD). 3. "City" სვეტში სახელები არეულია (მაგ: TbLisi, Tbilis, თბილისი, tbilisi). ყველა ჩაანაცვლე ერთი სტანდარტული ფორმით: "Tbilisi". იგივე გაუკეთე სხვა ქალაქებს მართლწერის მიხედვით. 4. "Full_Name" სვეტი გაყავი "First_Name" და "Last_Name" სვეტებად. მოქმედებების დასრულების შემდეგ, მომეცი ლოგი (Log) თუ რამდენი სტრიქონი წაიშალა და მომეცი სუფთა ფაილი გადმოსაწერად.

5. Excel & PowerBI: რთული ფორმულები, DAX და VBA მაკროსები

AI არ ჩაანაცვლებს ექსელს, ის აძლიერებს მას. თუ ცხრილის ატვირთვა არ გსურთ, გამოიყენეთ AI, რათა თქვენ მაგივრად დაწეროს ურთულესი სინტაქსი.

A. ფორმულების (XLOOKUP / INDEX-MATCH) გენერაცია

მთავარია ზუსტად აუხსნათ სად რა სვეტია განთავსებული.

მე მაქვს Excel-ის ფაილი ორი ფურცლით (Sheets): 'Employees' და 'Salaries'. - 'Employees' ფურცელზე, C სვეტში წერია პირადობის ნომერი. - 'Salaries' ფურცელზე, A სვეტში წერია პირადობის ნომერი, ხოლო E სვეტში შესაბამისი ხელფასი. მინდა, რომ 'Employees' ფურცლის D სვეტში ავტომატურად გაჩნდეს ხელფასი პირადობის მიხედვით. დამიწერე ყველაზე სწრაფი და ოპტიმალური ფორმულა. თუ ვინმე ვერ მოიძებნა, გამოაჩინე ტექსტი "ბაზაში არ არის".
=IFNA(XLOOKUP(C2, Salaries!A:A, Salaries!E:E), "ბაზაში არ არის")

B. PowerBI-სთვის DAX ფუნქციების დაწერა

Power BI მომხმარებლებისთვის DAX ფორმულების წერა ერთ-ერთი ყველაზე რთული ბარიერია.

// პრომპტი: დამიწერე DAX measure, რომელიც დათვლის გაყიდვებს მხოლოდ მიმდინარე თვისთვის, მაგრამ გამოტოვებს იმ პროდუქტებს, რომელთა სტატუსია "Inactive". Current_Month_Active_Sales = CALCULATE( SUM(Sales[Amount]), DATESMTD(Calendar[Date]), Products[Status] <> "Inactive" )

C. VBA Macros - ერთი ღილაკით ავტომატიზაცია

თუ ყოველ დილით გიწევთ ერთი და იგივე რეპორტების გადმოწერა, დაკოპირება, ფერების შეცვლა და დაფორმატება - სთხოვეთ მოდელს VBA კოდი.

დამიწერე Excel VBA მაკროსი "MorningFormat". მისი გაშვებისას მან უნდა: 1. წაშალოს პირველი 3 ზედმეტი სტრიქონი, 2. გააყინოს (Freeze Panes) პირველი სვეტი და მოწინავე სტრიქონი (Header), 3. Headers გაუკეთოს მუქი ლურჯი ფონი და თეთრი თამამი (Bold) ტექსტი, 4. ყველა უჯრაზე დააყენოს ავტომატური სიგანე (AutoFit). მითხარი ზუსტად სად დავაკოპირო ეს კოდი ექსელში.

6. Advanced Analytics: პროგნოზები (Forecasting) და ტრენდები

არ არის აუცილებელი იცოდეთ რა არის 'Linear Regression' ან 'ARIMA' მოდელი. მთავარია დაუსვათ ბიზნეს-კითხვა.

მონაცემთა "დაკითხვა" (Data Interrogation)

როცა ფაილს ატვირთავთ, გამოიყენეთ მარტივი, ადამიანური ფრაზები: "იპოვე ანომალიები", "რა არის მთავარი ფაქტორი?", "იწინასწარმეტყველე".

ატვირთულია კომპანიის ბოლო 3 წლის გაყიდვების დეტალური ბაზა. გთხოვ დაივიწყო ზედაპირული სტატისტიკა და ჩაუღრმავდე ბაზას. მიპასუხე შემდეგ 3 კითხვაზე: 1. Seasonality: ხედავ თუ არა რაიმე განმეორებად სეზონურობას? რომელ თვეებშია პიკი? 2. 80/20 წესი (Pareto): დამიმტკიცე ან უარყავი მუშაობს თუ არა პარეტოს პრინციპი ამ ბაზაზე (თუ მოაქვს კლიენტების 20%-ს შემოსავლის 80%). 3. Forecasting: ააგე პროგნოზირების მათემატიკური მოდელი და იწინასწარმეტყველე 2026 წლის Q1-ის მოსალოდნელი გაყიდვები 3 სცენარით (ოპტიმისტური, პესიმისტური, რეალისტური).

7. Claude Artifacts Masterclass: რეალური დროის ინტერაქტიული Dashboard-ები

ChatGPT Advanced Data Analysis არის საუკეთესო **გამოსათვლელად**, მაგრამ Claude 4.6 (Artifacts) არის საუკეთესო **წარსადგენად (Presentation)**.

Claude-ს შეუძლია თქვენი ცხრილი გადააქციოს ცოცხალ, მაუსზე რეაგირებად (Hover effects) და ლამაზ ვებ-აპლიკაციად (Dashboard).

მაგალითი: პროექტების მართვის Dashboard-ის გენერაცია

გამოიყენე თანდართული "Projects_2025.csv" ფაილის მონაცემები. გამოიყენე რეაქტი (React), Tailwind CSS და Recharts ბიბლიოთეკა და შექმენი სრულფასოვანი ინტერაქტიული Dashboard-ი დირექტორთა საბჭოსთვის. მოთხოვნები: 1. ზემოთ განათავსე 4 დიდი KPI ბარათი: - დასრულებული პროექტები (%) - ბიუჯეტის გადახარჯვა (ლარში) - რისკის ქვეშ მყოფი პროექტები - ჯამური ბიუჯეტი 2. ცენტრში დახატე ბარ-ჩარტი (Bar Chart), რომელიც აჩვენებს პროექტების სტატუსს დეპარტამენტების მიხედვით. 3. ბოლოში გააკეთე ცხრილი წითლად მონიშნული "Risky" პროექტებისთვის. 4. დიზაინი გააკეთე თანამედროვე "Dark Mode Glassmorphism" სტილში, თვალისმომჭრელი ნეონის ფერებით.

ამის გაშვებისთანავე, კლოდის მარჯვენა პანელში აეწყობა რეალური პროგრამული პროდუქტი, რომელიც შეგიძლიათ პირებზე (Stakeholders) აჩვენოთ.

8. რეალური ბიზნეს-სცენარები 5 სხვადასხვა ინდუსტრიისთვის (Cheat Sheet)

👨‍💼 HR / კადრები

"ატვირთულია თანამშრომელთა ბაზა (ასაკი, ხელფასი, სტაჟი, კმაყოფილება, წასვლის სტატუსი). გამოიყენე Decision Tree ალგორითმი და გამომითვალე, რომელი 3 ყველაზე მნიშვნელოვანი ფაქტორი განაპირობებს თანამშრომლის კომპანიიდან წასვლას. შექმენი პროფილის პორტრეტი თანამშრომლისა, რომელიც წასვლის ყველაზე დიდ რისკ-ჯგუფშია."

🛒 E-Commerce & საცალო გაყიდვები

"ბაზაშია 1 წლის ონლაინ გაყიდვების ჩეკები (Order_ID, Product_Name). ჩაატარე Market Basket Analysis (Market Affinity). მაჩვენე ტოპ 5 პროდუქტების წყვილი, რომლებსაც კლიენტები ერთად ყიდულობენ ყველაზე ხშირად. ეს მჭირდება საიტზე "Recommended Products"-ის დასაყენებლად."

🎯 მარკეტინგი

"ატვირთულია სარეკლამო კამპანიების მონაცემები. გამოთვალე თითოეული არხის (Facebook, Google, TikTok) ROI, CPA და 1 შემოსულ ლიდზე დანახარჯი. შექმენი Scatter Plot ვიზუალიზაცია: `Y-ღერძზე` დახარჯული ბიუჯეტი, `X-ღერძზე` მოტანილი კონვერსიები. ბუშტის (Bubble) ზომა იყოს ჯამური ROI."

🚚 ლოგისტიკა და მარაგები

"ბაზაშია არსებული მარაგები, მათი მოხმარების დღიური ტემპი (Burn rate) და მწარმოებლიდან მოწოდების დრო (Lead time). გამომითვალე თითოეული პროდუქტისთვის ზუსტად როდის უნდა გავაკეთო ახალი შეკვეთა (Reorder Point), რომ მარაგი არ გამინულდეს. მონიშნე წითლად კრიტიკული SKU-ები."

📈 ფინანსები

9. რომელი მოდელი გამოვიყენოთ? (GPT-5.4 vs Claude 4.6 vs Perplexity)

ამოცანა საუკეთესო ინსტრუმენტი რატომ?
**დიდი CSV ფაილების ატვირთვა და სტატისტიკური გამოთვლები** ChatGPT (Adv. Data Analysis) მუშაობს Python გარემოში. იძლევა მათემატიკურად 100%-იან სიზუსტეს. გამოიმუშავებს ექსელის ფაილებს უკან გადმოსაწერად.
**რთული Excel VLOOKUP/VBA ფორმულების დაწერა** Claude 3.5 Sonnet / 4.6 საუკეთესოდ კითხულობს და წერს კოდს/სინტაქსს. კარგად ესმის კომპლექსური ლოგიკა.
**მენეჯმენტთან წარსადგენი ინტერაქტიული Dashboard-ის შექმნა** Claude 4.6 Artifacts React-ის და Recharts-ის დახმარებით მარჯვენა ეკრანზე თავად ხატავს UI ინტერფეისებს აპლიკაციად.
**ბაზრის (მაკროეკონომიკური) მონაცემების მოძიება გარედან** Perplexity Pro პირდაპირ წვდება ცოცხალ ვებს, კითხულობს ოფიციალურ PDFs და შეძლებს მოპოვებული ინფორმაციით პირდაპირ ცხრილის შექმნას.
