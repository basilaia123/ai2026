🔄 Iterative Refinement Prompts (გამოიყენეთ თანმიმდევრულად):

იტერაცია 1 - CTA

დაამატე Hero Section-ში "Get Started" ღილაკი, რომელიც ხსნის Email Sign-up მოდალს. ღილაკი უნდა იყოს ყვითელი/ნარინჯისფერი და გამორჩეული.

იტერაცია 2 - სოციალური მტკიცებულება

დაამატე "Trusted by" სექცია Hero-ს ქვემოთ - 4-5 ლოგო (placeholder-ებით, მოგვიანებით შეცვლით).

იტერაცია 3 - FAQ

დაამატე FAQ სექცია Testimonials-ის ქვემოთ - 5 ხშირი კითხვა-პასუხი, accordion-style.

იტერაცია 4 - Email Sign-up + Supabase

დაამატე Email Sign-up ფორმა Footer-ში: "Join 1,000+ early adopters" + email input + "Subscribe" ღილაკი. მონაცემები უნდა შეინახოს Supabase-ის "subscribers" ცხრილში. Lovable-მ თავად დააკონფიგურიროს Supabase integration.

⚡ სავარჯიშო 5: Automation Workflow (Make.com)

⏱️ 30 წუთი 📊 დონე: საშუალო-რთული 🎯 ბლოკი IV

📋 დავალება:

Make.com-ში შექმენით Workflow, რომელიც Landing Page-დან მიღებულ Lead-ს ავტომატურად დაამატებს Google Sheets-ში და გამოაგზავნის Welcome Email-ს.

📋 3 მოდული:

  1. Trigger: Webhooks → Custom Webhook (POST request from Lovable form)
  2. Action 1: Google Sheets → Add a Row (Email, Name, Timestamp)
  3. Action 2: Email → Send an Email (Welcome message template)

🔧 Setup Steps:

Make.com Setup Guide

1. რეგისტრაცია: make.com → Sign up → Choose Free Plan

2. Create Scenario: + Create a new scenario → Search "Webhooks" → "Custom webhook" → Add → Copy webhook URL

3. Lovable-ში: Email form submission-ის code-ში დაამატე fetch() POST request to webhook URL

4. Test Trigger: Submit form on Lovable once → ნახეთ Make.com-ში "1 bundle received"

5. Add Google Sheets Module: + → Google Sheets → "Add a Row" → Connect account → Choose spreadsheet + sheet

6. Add Email Module: + → Email → "Send an Email" → Configure SMTP (Gmail app password) or use built-in

7. Email Template: "გამარჯობა {{1.name}}! მადლობა, რომ გამოგვწერეთ. მოგვიანებით დაგიკავშირდებით."

8. Activate: Save → "Scheduling: ON" → Submit test form → Verify Sheet + Email
