🗄️ Supabase ინტეგრაცია

რას გაძლევთ:

  • Database: PostgreSQL
  • Auth: რეგისტრაცია/ავტორიზაცია
  • Storage: ფაილების შენახვა
  • Real-time: ცოცხალი მონაცემები

Lovable-ში დასამატებლად:

"Connect to Supabase and create a users table. Add authentication with email/password. Create a dashboard that shows user data."

✅ DO (გააკეთე)

  • ✅ დაიწყე მარტივით
  • ✅ კონკრეტული პრომპტები
  • ✅ იტერაციული დახვეწა
  • ✅ Mobile-first დიზაინი
  • ✅ GitHub Export რეგულარულად
  • ✅ რეფერენსები ("like Stripe")
  • ✅ "shadcn/ui components"
  • ✅ ტესტირება ბრაუზერებში

❌ DON'T (არ გააკეთო)

  • ❌ ბუნდოვანი პრომპტები
  • ❌ ყველაფერი ერთ პრომპტში
  • ❌ კოდის უგულებელყოფა
  • ❌ Performance-ის არშემოწმება
  • ❌ Mobile-ის დავიწყება
  • ❌ AI-ს 100% ენდობა
  • ❌ Security-ის იგნორირება
  • ❌ Backup-ის არქონა

⚠️ ხშირი პრობლემები და გადაწყვეტილებები

❌ AI არ იგებს რა მინდა
✅ გამოსავალი: დეტალური პრომპტი + სტრუქტურა + ფერები + რეფერენსი
❌ შედეგი არ მომწონს
✅ გამოსავალი: იტერაცია - კონკრეტული ცვლილებები ცალ-ცალკე
❌ Mobile-ზე ცუდად გამოიყურება
✅ გამოსავალი: "გააკეთე mobile-responsive" + "Create hamburger menu"
❌ Supabase არ უკავშირდება
✅ გამოსავალი: შეამოწმე API Keys პარამეტრებში
