📚 ლექცია 5: შეჯამება

ვიზუალური კონტენტი და პრეზენტაციები

მოკლე კონსპექტი v2 სლაიდების მიხედვით

🎯 ლექციის საერთო სურათი

ეს ლექცია ფოკუსირებულია AI-ით ვიზუალური კონტენტის სწრაფ და პროფესიულ შექმნაზე: გამოსახულებების გენერირება, prompt engineering, visual consistency, AI პრეზენტაციები, ბრენდის ერთიანობა, ქართული ტიპოგრაფია და ეთიკა.

ძირითადი ბლოკები:

  • 2026 წლის წამყვანი image tools;
  • ძლიერი prompt-ის ფორმულა;
  • reference-based consistency Gemini / ChatGPT / Midjourney-ში;
  • Gamma workflow outline-იდან deck-მდე;
  • design principles, storytelling, brand consistency;
  • ქართული ტექსტის და ეთიკის პრაქტიკული საკითხები.

🎨 ძირითადი AI ინსტრუმენტები

Nano Banana 2 (Gemini 3.1 Flash Image)

  • ყველაზე სწრაფი all-around image tool;
  • კარგია სწრაფი კამპანიის ვარიაციებისა და mockup-ებისთვის;
  • ძლიერია instruction following-სა და consistency-ზე.

Nano Banana Pro (Gemini 3 Pro Image)

  • უფრო polished და studio-grade შედეგები;
  • კარგია პოსტერების, შეფუთვის, diagrams-ისა და ბრენდული ვიზუალებისთვის;
  • ძლიერია ტექსტთან და რთულ სცენებთან.

GPT Image 2 / ChatGPT Images 2.0

  • საუკეთესო conversational editing;
  • კარგი text rendering და არსებული image input-ის ტრანსფორმაცია;
  • ქართული მარკეტინგული ვიზუალებისთვის ერთ-ერთი ყველაზე პრაქტიკული არჩევანი.

Midjourney v7

  • ძლიერი სტილი და cinematic ესთეტიკა;
  • კარგია moodboards, luxury visuals და creative მიმართულებისთვის;
  • აქვს explicit consistency controls: --sref, --cref.

✍️ Prompt Engineering

მთავარი ფორმულა

Subject + Action + Environment + Style + Lighting + Camera/Format + Constraints

რას ნიშნავს თითო ნაწილი

  • Subject: ვინ ან რა არის მთავარ კადრში;
  • Action: რას აკეთებს ან რა პოზაშია;
  • Environment: სად ხდება სცენა;
  • Style: ფოტორეალისტური, cinematic, minimalist და ა.შ.;
  • Lighting: natural window light, golden hour, studio lighting;
  • Camera/Format: close-up, medium shot, 16:9, 4:5;
  • Constraints: რა უნდა დარჩეს უცვლელი ან რა უნდა შეიცვალოს მხოლოდ.

სუსტი vs ძლიერი

სუსტი: „ქალი ოფისში“

ძლიერი: „Georgian startup founder, presenting to a team, modern glass office, premium realistic photography, soft natural window light, medium shot, 16:9, keep the same face and outfit, change only the background“

🔄 Visual Consistency

Midjourney

  • --sref ინარჩუნებს სტილს, ფერებს, ატმოსფეროს;
  • --cref ინარჩუნებს პერსონაჟს/ობიექტს სხვადასხვა სცენაში.

Gemini / ChatGPT Images

  • პირველი სურათი შექმენით Prompt 1-ით;
  • შემდეგ მოითხოვეთ: Use the first image as the character reference;
  • Prompt 2-ში მკაფიოდ გაყავით: რა დარჩეს იგივე და რა შეიცვალოს.

უნივერსალური წესი

  • Lock attributes: face, hair, proportions, outfit identity, palette;
  • Change only: background, setting, pose, camera angle.

📊 AI პრეზენტაციები

Gamma AI

  • სწრაფი deck generation;
  • ვიზუალური კოჰერენცია;
  • მომზადებული outline-ის ჩასმა საუკეთესო გზაა.

საუკეთესო workflow

  1. Gemini/ChatGPT-ში მოამზადეთ მკაფიო outline;
  2. მიუთითეთ თემა, აუდიტორია, tone, slide count, storytelling structure;
  3. Gamma-ში გამოიყენეთ Paste Text რეჟიმი;
  4. ხელით დახვეწეთ headings, visuals და pacing.

Prompt pattern

იმოქმედე როგორც პრემიუმ პრეზენტაციების დიზაინერმა და კოპირაიტერმა... გამოიყენე Storytelling (პრობლემა > გამოსავალი)... თითოეული სლაიდისთვის დამიწერე Hook, 3 ბულეტი, ვიზუალის იდეა.

🎨 დიზაინი, Storytelling და ბრენდი

დიზაინის ძირითადი პრინციპები

  • Contrast;
  • Alignment;
  • Proximity;
  • White Space;
  • Hierarchy.

Storytelling ელემენტები

  • Character: ვინ არის მთავარი გმირი;
  • Conflict: რა პრობლემაა;
  • Context: სად და როდის ხდება მოქმედება;
  • Resolution: როგორ გვარდება პრობლემა.

Brand Consistency

  • ერთიანი ფერთა პალიტრა;
  • ერთიანი ტიპოგრაფია;
  • სტაბილური ფოტო-სტილი და განათება;
  • ერთიანი ვიზუალური ენა ყველა asset-ში.

🇬🇪 ქართული ტიპოგრაფია

პრაქტიკული წესი

  • ვიზუალი გენერირება AI-ში;
  • ქართული headline / slogan / ფასი დაამატეთ Canva, Figma ან Photoshop-ში;
  • გამოიყენეთ გამართული ქართული ფონტები, მაგალითად FiraGO ან BPG.

⚖️ ეთიკა და კომერციული გამოყენება

  • არ მოახდინოთ ცოცხალი მხატვრის სტილის პირდაპირი იმიტაცია;
  • არ შექმნათ misleading visuals ან deepfake-ები;
  • ყოველთვის გადაამოწმეთ tool-ის კომერციული terms;
  • ბრენდის და რეალური ადამიანების გამოყენებისას იმოქმედეთ პასუხისმგებლობით.

⚙️ პრაქტიკული Workflow

  1. Planning: მიზანი, აუდიტორია, format;
  2. Ideation: mood board, references, prompt drafting;
  3. Creation: image generation ან AI deck;
  4. Refinement: brand check, text cleanup, export;
  5. Publishing: platform-specific ზომები და final output.

✨ საუკეთესო პრაქტიკა და ხშირი შეცდომები

გააკეთეთ

  • იყავით კონკრეტული prompt-ებში;
  • შექმენით prompt library;
  • გამოიყენეთ references consistency-ისთვის;
  • ტექსტიანი ვიზუალებისთვის გადაამოწმეთ საბოლოო rendering.

არ გააკეთოთ

  • არ ენდოთ AI-ს ქართულ ტექსტს უსაბუთოდ;
  • არ გადაავსოთ სლაიდები ელემენტებით;
  • არ გამოიყენოთ შემთხვევითი სტილი ერთსა და იმავე ბრენდში;
  • არ დაკარგოთ დრო თავიდან prompt-ების წერაში, თუ შეგეძლებათ მათი ბიბლიოთეკაში შენახვა.

🎯 ძირითადი დასკვნები

  • AI პროფესიულ ვიზუალს ყველასთვის ხელმისაწვდომს ხდის;
  • Prompt engineering არის შედეგის ხარისხის მთავარი ბერკეტი;
  • Consistency არის ბრენდული ხარისხის აუცილებელი ნაწილი;
  • Gamma-სთვის საუკეთესო deck იწყება კარგი outline-ით;
  • ქართული ტექსტი უმეტესად ჯობს ხელით დაემატოს;
  • Prompt library და brand guide მნიშვნელოვნად ზოგავს დროს.
