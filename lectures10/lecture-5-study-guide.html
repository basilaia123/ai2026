🎨 ლექცია 5: ვიზუალური კონტენტი და პრეზენტაციები

სასწავლო გზამკვლევი

🏠 მთავარი 📊 სლაიდები 📝 შეჯამება 📚 სასწავლო გზამკვლევი ⚡ სწრაფი ცნობარი 🎯 სავარჯიშოები

🎯 ამ ლექციის მიზანი

ლექციის ბოლოს თქვენ უნდა შეგეძლოთ:

  • 2026 წლის მთავარი AI image tools-ის გარჩევა: Nano Banana 2, Nano Banana Pro, GPT Image 2, Midjourney v7;
  • ძლიერი prompt-ის აგება ფორმულით: Subject + Action + Environment + Style + Lighting + Camera/Format + Constraints;
  • reference image-ების გამოყენებით ერთი და იგივე პერსონის ან პროდუქტის განმეორება;
  • LLM-ის დახმარებით Gamma-სთვის პრეზენტაციის მკაფიო outline-ის მომზადება;
  • ქართული ტიპოგრაფიის, ბრენდის ერთიანობისა და ეთიკის საკითხების სწორად მართვა.

📋 როგორ მოემზადოთ

ინსტრუმენტები, რომლებიც ჯობს გქონდეთ მზად

  • ერთი მაინც image tool: ChatGPT Images, Gemini ან Midjourney;
  • ერთი presentation tool: Gamma ან Canva;
  • ერთი საქაღალდე prompt-ებისა და reference image-ებისთვის;
  • 2-3 რეალური ქეისი: პროდუქტი, პორტრეტი, სოციალური მედიის ვიზუალი ან კაფეს კამპანია.

რა დააკვირდეთ სლაიდების ყურებისას

  • რომელი tool არის სწრაფი და რომელი უფრო ზუსტი ტექსტთან ან ბრენდულ გამოსახულებასთან;
  • როგორ იცვლება შედეგი, როცა prompt-ს ემატება camera/format/constraints;
  • როგორ მუშაობს consistency Midjourney-ში პარამეტრებით და Gemini/ChatGPT-ში reference image-ებით;
  • რატომ ჯობს Gamma-ს outline ჯერ LLM-ში მოამზადოთ და არა პირდაპირ presentation tool-ში;
  • როგორ უნდა იმუშაოთ ქართულ ტექსტთან, როცა AI ასოებს არასწორად აჩვენებს.

🧭 სწავლების რუკა

1. Tool Landscape

სლაიდები 4-5 აჩვენებს, რომ ყველა ინსტრუმენტი ერთნაირი არ არის. Nano Banana 2 კარგია სწრაფი iteration-ისთვის, Nano Banana Pro და GPT Image 2 გამორჩეულია ტექსტთან და refined output-თან, Midjourney კი ძლიერი რჩება მხატვრულ სტილში. Gamma AI ლიდერობს სწრაფ AI presentations-ში.

2. Prompt Engineering

სლაიდები 6-11 ლექციის ბირთვია. აქ სწავლობთ არა უბრალოდ „რა დავწერო“, არამედ როგორ ააწყოთ prompt პროფესიულად, როგორ მიუთითოთ სტილი, განათება, კადრი, ფორმატი და constraints.

3. Consistency Workflow

სლაიდი 12 გიჩვენებთ სამ განსხვავებულ გზას: Midjourney-ში --sref / --cref, Gemini-ში uploaded reference image, ChatGPT Images-ში იგივე reference + conversational editing.

4. Presentation Workflow

სლაიდები 14-16 გასწავლით, როგორ მოამზადოთ Premium presentation outline Gemini/ChatGPT-ში და შემდეგ გადაიტანოთ Gamma-ში. აქ განსაკუთრებით მნიშვნელოვანია აუდიტორიის, tone-ის და storytelling structure-ის მითითება.

5. Brand, Typography, Ethics

სლაიდები 18-24 გეუბნებათ, რომ კარგი ვიზუალი მხოლოდ generative შედეგი არ არის. საჭიროა storytelling, ბრენდის ერთიანობა, ქართული ტექსტის ხელით კორექცია და ეთიკური ჩარჩო.

🧪 ლექციის შემდეგ პრაქტიკა

სავარჯიშო A: Prompt Upgrade

  1. აირჩიეთ სუსტი იდეა, მაგალითად: „ქალი ოფისში“.
  2. გადააკეთეთ იგი სრულ prompt-ად ფორმულით.
  3. გაუშვით სუსტი და ძლიერი ვერსია და შეადარეთ შედეგები.

სავარჯიშო B: Same Person, New Setting

  1. Prompt 1-ით შექმენით პერსონა.
  2. Prompt 2-ში მოითხოვეთ იგივე პერსონა სხვა გარემოში.
  3. უცვლელი დატოვეთ face / outfit / style, შეცვალეთ მხოლოდ setting / angle / pose.

სავარჯიშო C: Gamma Outline

  1. Gemini ან ChatGPT-ს მიეცით prompt: იმოქმედე როგორც პრემიუმ პრეზენტაციების დიზაინერმა და კოპირაიტერმა.
  2. მოითხოვეთ 6-სლაიდიანი სტრუქტურა ახალი ყავის კაფის თემაზე, პრობლემიდან გამოსავლამდე.
  3. მიღებული outline ჩასვით Gamma-ში და შეადარეთ საბოლოო deck-ს.

🇬🇪 ქართული კონტექსტი

რაც განსაკუთრებით უნდა დაიმახსოვროთ

  • ChatGPT Images და Gemini ქართულ prompt-ს კარგად იგებენ, მაგრამ გამოსახულებაში ქართული ასოების ზუსტი რენდერინგი მაინც პრობლემა შეიძლება იყოს;
  • თუ გჭირდებათ გამართული headline, ფასი ან slogan, უმჯობესია ტექსტი ხელით დაამატოთ Canva / Figma / Photoshop-ში;
  • ლოკალური ვიზუალისთვის მნიშვნელოვანია კონტექსტის სწორად აღწერა: ძველი თბილისი, აგურის ტექსტურა, ხის აბრები, ადგილობრივი ატმოსფერო.

🎓 თვითშეფასება

  • ვიცი, როდის ავირჩიო Nano Banana 2 და როდის GPT Image 2 ან Midjourney;
  • შემიძლია prompt-ის ფორმულის გამოყენება რეალურ მაგალითზე;
  • ვიცი reference-based consistency-ის მინიმუმ ერთი მეთოდი;
  • ვიცი, როგორ მოვამზადო outline Gamma-სთვის Gemini/ChatGPT-ში;
  • ვიცი, როგორ მოვიქცე, თუ AI ქართულ ტექსტს არასწორად გენერირებს;
  • ვიცი, რომ კომერციულ გამოყენებამდე ეთიკა და terms უნდა გადავამოწმო.
