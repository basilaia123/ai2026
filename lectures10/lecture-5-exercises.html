🎨 სავარჯიშო 1: Prompt Upgrade

ხანგრძლივობა: 15-20 წუთი | რთულება: ⭐⭐

დავალება

აიღეთ 3 სუსტი prompt და გადააკეთეთ პროფესიულ prompt-ად ლექციის ფორმულის გამოყენებით.

სუსტი იდეები

  • „ქალი ოფისში“
  • „ყავის რეკლამა“
  • „ყურსასმენები“

უნდა დაამატოთ

  • Subject
  • Action
  • Environment
  • Style
  • Lighting
  • Camera/Format
  • Constraints

🔄 სავარჯიშო 2: იგივე პერსონა, ახალი გარემო

ხანგრძლივობა: 20-25 წუთი | რთულება: ⭐⭐⭐

დავალება

შექმენით ორი დაკავშირებული სურათი. Prompt 1 ქმნის პერსონას, Prompt 2 იმავე პერსონას აჩვენებს სხვა setting-ში.

მინიმალური სტრუქტურა

  1. Prompt 1: შექმენით საწყისი პერსონა ან პროდუქტი.
  2. Prompt 2: გამოიყენეთ პირველი სურათი reference-ად.
  3. Prompt 2-ში მკაფიოდ დაწერეთ: რა დარჩეს იგივე და რა შეიცვალოს.

მიუთითეთ პირდაპირ

  • Use the first image as the character reference
  • Keep the same face / hair / outfit identity
  • Change only the background / pose / camera angle

🛍️ სავარჯიშო 3: კაფეს 3 ვიზუალი

ხანგრძლივობა: 20 წუთი | რთულება: ⭐⭐⭐

დავალება

გაიმეორეთ ლექციის კაფეს სცენარი და შექმენით 3 visuals Instagram-ისთვის.

აუცილებელი 3 კადრი

  1. პროდუქტის ფოტო (კაპუჩინო / დესერტი / სასმელი)
  2. ინტერიერის ატმოსფერო
  3. გარე ხედი ან storefront

დამატებითი პირობა

  • ერთი prompt იყოს ინგლისურად;
  • ერთი prompt იყოს ქართულად;
  • შეადარეთ რომელ tool-ს უკეთ ესმის ლოკალური ატმოსფერო.
