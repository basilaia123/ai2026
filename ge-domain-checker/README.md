# .ge Domain Checker

ვებ აპი, რომელიც სიიდან ამოწმებს `.ge` დომენების ხელმისაწვდომობას `whois.nic.ge`-ზე.

## გაშვება

```bash
cd ge-domain-checker
npm start
```

შემდეგ გახსენი:

`http://localhost:3000`

## როგორ მუშაობს

- UI-ში ჩასვი დომენების სია (თითო ახალ ხაზზე).
- შესაძლებელია `mybrand` ან `mybrand.ge` ფორმატი.
- Backend თითოეულ დომენზე აგზავნის WHOIS მოთხოვნას და აბრუნებს სტატუსს:
  - `available` - თავისუფალია
  - `taken` - დაკავებულია
  - `unknown` - პასუხი ვერ განისაზღვრა
  - `invalid` - დომენის ფორმატი არასწორია
  - `error` - ქსელური/WHOIS შეცდომა

## დამატებული ფუნქციები

- დომენის გენერატორი (`20/30/40/50` იდეა საკვანძო სიტყვიდან)
- `Premium Score` (0-100) + tier (`premium/strong/good/average/weak`)
- `CSV Export` და `Excel Export (.xls)`
- ფავორიტები: ვარსკვლავით მონიშვნა, კოპირება და სიაში ავტომატური ჩამატება

## შენიშვნა

აპს ინტერნეტთან წვდომა სჭირდება, რადგან შემოწმება რეალურ დროში ხდება WHOIS სერვერზე.
