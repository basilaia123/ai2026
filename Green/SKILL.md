## 2. Proposal Synthesis Workflow

When requested to draft a proposal or training syllabus, strictly execute these chronological steps:

1. **Client Profiling & Goal Definition**:
   - Assess the industry sector. Match it to one of the 5 database frameworks above, or construct an analogous corporate system.
   - Establish the target goals: time savings on admin tasks (standard target: 40-60%), automated SOP sheets, personal agents, or marketing assets.

2. **Session and Hour Architecture**:
   - Standard structure is **3-hour sessions** or **2.5-hour sessions** (typically 3, 6, or 8 sessions total).
   - Divide each session into precise 30-45 minute blocks with a dedicated 10-15 minute coffee break (`☕ შესვენება`).
   - Define exact practical assignments (`📋 საშინაო დავალება`) for each session.

3. **Color and Typography Theme Alignment**:
   - Recommend the visual colors and fonts that match the brand archetype:
     - *Green School* → Eco Green Theme
     - *Credo Bank* → Credo Navy & Orange Theme
     - *Gastronome / Tempo* → Luxury Charcoal & Gold Theme
     - *ELT Group* → Hospitality Gold & Navy Theme
     - *Smart Academy / GITA* → Tech Indigo Theme
     - *Art / Design* → Creative Coral Theme

4. **Detailed Core Module Assembly**:
   Generate detailed descriptions, bullets, and exercise outlines in Georgian using the following modular index:
   - **Module 1**: AI-ს არსი, ქართული ენა და აგენტური ეფექტურობა (Core Prompting, CREATE formula, Georgian syntax rules).
   - **Module 2**: ბიზნეს-კომუნიკაცია და ელფოსტის ავტომატიზაცია (B2B emails, diplomatic tones, Meeting Minutes summaries).
   - **Module 3**: საოპერაციო SOP-ები და ვიზუალური პრეზენტაციები (Standard operating procedures, Canva/Gamma AI slides).
   - **Module 4**: გაყიდვები, Objection Handling და კლიენტის სკრიპტები (Upselling, objections playbooks, customer personas).
   - **Module 5**: Excel და მონაცემთა ანალიზი AI-ის დახმარებით (Formulas, ROP inventory calculators, route optimizations).
   - **Module 6**: პერსონალური AI ასისტენტები და Deep Research (Custom GPTs, Gems, Claude Projects setup, Perplexity, NotebookLM).
   - **Module 7**: n8n / Make.com ავტომატიზაციები და No-Code აპლიკაციები (Workflows, Lovable.app prototyping).
   - **Module 8**: კომპანიის AI პოლიტიკა, GDPR და მონაცემთა უსაფრთხოება (GDPR 2026 compliance, safety rules, 30-day rollout roadmap).

## 3. Strict Georgian Syntax & Formatting Rules (Linguistic Standard)

To match the high-end editorial quality of your courses, you MUST strictly enforce the following grammatical rules for all Georgian outputs:

1. **Georgian Quotation Marks strictly**:
   - Use double bottom-top quotes: `,,...“` or `„...“` for all Georgian textual references.
   - **CRITICAL**: Never modify standard computer quotes (`"..."` or `'...'`) inside HTML tags, code attributes, or programming classes.

2. **Hyphen and Dash Protocols**:
   - Completely BAN both em-dashes (`—`) and en-dashes (`–`). They are grammatically incorrect in Georgian.
   - Replace all sentence dividers and compound connectors with a standard single hyphen: `-`.
   - Examples: `3-სესიიანი`, `10-ჯერ`, `2-3-საათიანი`.

3. **Georgian Case Attachment on English Terms / Acronyms**:
   - Always attach case suffixes using a hyphen: `AI-ს`, `AI-ით`, `Notion-ში`, `Excel-ის`, `Make.com-ში`.
   - Phonetic Rule for AI: Because "AI" is pronounced [ei-ai], it ends with the vowel "i". Therefore, its genitive form MUST be written as `AI-ის` (not `AI-ს`).
   - Suffix Agent of Action Suffix: Never use genitive pronouns with `-ს` in agent passive constructions (e.g., write `თქვენ მიერ` instead of `თქვენს მიერ`).

4. **Strict Prohibition of Abbreviations**:
   - Fully expand all abbreviations. Do not write `მაგ.`, `სთ.`, `გვ.`, `შპს`.
   - Write: `მაგალითად:`, `საათი`, `გვერდი`, `შეზღუდული პასუხისმგებლობის საზოგადოება`.

5. **Barbarism Replacement (Corporate Jargon to Georgian)**:
   - Convert corporate office slang to correct Georgian:
     - `თასქი` → `დავალება`
     - `დეფოლტად` → `სტანდარტულად`
     - `ქეისი` → `შემთხვევა` / `ქეისი` (depending on whether it's business case-study)
     - `სედულერი` → `განრიგი`
