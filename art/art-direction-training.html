Prompt Engineering - AI-სთან სწორი კომუნიკაცია

როგორ დავუსვათ AI-ს კითხვები საუკეთესო შედეგისთვის. ცუდი და კარგი პრომპტის შედარება ქართულად, შემდეგ კი live refinement-ის რამდენიმე ეტაპი.

❌ ცუდი პრომპტი:

"დამიწერე ტექსტი ჩვენი კონცერტის შესახებ"

AI-მ არ იცის: ვის, რა ტონით, რა ფორმატში, რა სიგრძით და რა მიზნისთვის

✓ კარგი პრომპტი:

"შენ ხარ კულტურული ორგანიზაციის სოციალური მედიის მენეჯერი..."

მოიცავს: როლს, კონტექსტს, ფორმატს, სამიზნე აუდიტორიას

R

"შენ ხარ გამოცდილი კოპირაიტერი, სპეციალიზებული კულტურულ ღონისძიებებში"

C

Context - კონტექსტი

ვინ ვართ, რა ვაკეთებთ, ვისთვის, სად

T

Task - დავალება

კონკრეტული, გაზომვადი მოთხოვნა

F

Format - ფორმატი

სიგრძე, სტილი, სტრუქტურა, ენა

პრომპტი - RCTF მეთოდით (სწრაფი ვერსია):

ChatGPT / Claude-ისთვის - სწრაფი prompt 
Role:
შენ ხარ კულტურული ორგანიზაციის კომუნიკაციის სპეციალისტი.

Context:

- სიგრძე: 150-200 სიტყვა
- ტონი: ენთუზიასტური, მეგობრული, პროფესიონალური
- მოიცავდეს: სათაურს, ძირითად ინფორმაციას, CTA-ს
- ქართულად

CREATE ფორმულა - უფრო ძლიერი ჩარჩო პრომპტის დასაწერად

ამ ტრენინგში `CREATE`-ს ვიყენებთ როგორც მარტივ სამუშაო ფორმულას: Context, Role, Example, Action, Tone, Expected Output.

7-10 წთ

C - Context

რა პროექტია, ვისთვის კეთდება, რა სიტუაციაში ვიყენებთ.

R - Role

ვინ უნდა იყოს AI: კოპირაიტერი, კურატორი, კომუნიკაციის მენეჯერი.

E - Example

ერთი მოკლე ნიმუში ან სასურველი სტილის აღწერა, თუ გაქვთ.

A - Action

კონკრეტულად რა უნდა გააკეთოს ახლა: დაწეროს, შეადაროს, გადააკეთოს.

T - Tone

ოფიციალური, თბილი, ენერგიული, აკადემიური ან ახალგაზრდული.

E - Expected Output

რა ფორმით უნდა დააბრუნოს პასუხი: სიგრძე, ენა, სექციები, bullets, CTA.

პრომპტი - CREATE ფორმულით (სადემონსტრაციო მაგალითი):

ChatGPT / Claude-ისთვის 
Context:
ჩვენ ვართ [ორგანიზაცია], ვაწყობთ [ღონისძიება] - [თარიღი], [ადგილი].
სამიზნე აუდიტორია: [ვინ].

Role:
შენ ხარ კულტურული ორგანიზაციის კომუნიკაციის სპეციალისტი.

Example:
ტექსტი უნდა ჰგავდეს თანამედროვე კულტურული ცენტრის Facebook პოსტს:
მკაფიო შესავალი, პრაქტიკული ინფორმაცია, თბილი მოწვევა.

Action:
დამიწერე Facebook-ის ღონისძიების აღწერა.

Tone:
ენთუზიასტური, მეგობრული, პროფესიონალური.

Expected Output:
- სიგრძე: 150-200 სიტყვა
- მოიცავდეს: სათაურს, ძირითად ინფორმაციას, CTA-ს
- ქართულად
- ბოლოს ცალკე გამოყავი 1-წინადადებიანი მოკლე ვერსია poster caption-ისთვის

როდის გამოვიყენო CREATE?

  • როცა პირველი პასუხი ზედმეტად ზოგადია
  • როცა გინდა ერთ prompt-ში ხარისხი და სტრუქტურა ორივე მოიგო
  • როცა გუნდისთვის reusable template-ის შექმნა გჭირდება

თუ დრო ცოტაა

  • სწრაფი ვერსია: `C + R + A + E` საკმარისია 2-3 წუთში
  • სრული ვერსია: `CREATE` დემო + ერთი live rewrite დაახლოებით 7-10 წუთი
  • მონაწილეების სავარჯიშოთი ეს ნაწილი იზრდება დაახლოებით 10-12 წუთამდე

CREATE vs RCTF - როდის რომელი?

RCTF

სწრაფი, მარტივი და კარგი ყოველდღიური ტექსტური დავალებებისთვის. 2-3 წუთში ასახსნელია.

CREATE

უფრო გაშლილი ჩარჩოა, როცა ხარისხი, ტონი და სასურველი შედეგის ფორმა უფრო ზუსტად უნდა მართო.

თანმიმდევრული დიალოგი - შემდგომი პრომპტები:

ნებისმიერი AI-სთვის - პასუხის შემდეგ 
გადააკეთე ეს ტექსტი: გახადე უფრო ოფიციალური და მოკლე (100 სიტყვამდე).

ახლა შექმენი მისი ინგლისური ვერსიაც.

ასევე მოამზადე Instagram-ის ვერსია, რომელიც მოიცავს 5 ჰეშთეგს.
