## 🗝️ მთავარი კონცეფციები (უნდა იცოდეთ)

### ტექსტის გენერაცია

**Definition:**
> ტექსტის გენერაცია არის პროცესი, რომლის დროსაც AI ქმნის ორიგინალურ, სტრუქტურირებულ და მიზანმიმართულ ტექსტს.

**Key Points:**
- AI არის assistant, არა replacement
- Output always needs editing
- Context და specificity matters
- Different tools for different needs

### კონტენტის ტიპები

**უნდა იცოდეთ:**

| ტიპი | მიზანი | მაგალითები |
|------|--------|------------|
| Informational | განათლება | Blog posts, tutorials |
| Business | პროფესიონალური კომუნიკაცია | Reports, proposals |
| Social Media | Engagement | Posts, stories, threads |
| Marketing | Conversion | Ads, landing pages |
| Creative | გასართობი/შთაგონება | Stories, poems |
| Technical | ტექნიკური info | Docs, API guides |

### ტონი და სტილი

**6 ძირითადი ტონი:**

1. **ფორმალური** - academic, legal, corporate
2. **პროფესიონალური** - business communications
3. **მეგობრული** - blogs, newsletters
4. **ტექნიკური** - documentation
5. **მოტივაციური** - speeches, presentations
6. **დამაჯერებელი** - sales, advertising

**მნიშვნელოვანი:** ტონი უნდა შეესაბამებოდეს აუდიტორიას და კონტექსტს!

### აუდიტორიის ადაპტაცია

**3 Key დონე:**

1. **Beginners**
   - მარტივი ენა
   - ანალოგიები
   - Step-by-step
   - Encouraging

2. **Professionals**
   - Industry terminology OK
   - Focus on insights
   - Data-driven
   - Actionable

3. **Executives**
   - High-level overview
   - Business impact
   - ROI focus
   - Brief და concise

### Copywriting ფორმულები

**AIDA:**
```
A - Attention (გაიპყარე ყურადღება)
I - Interest (დააინტერესე)
D - Desire (შექმენი სურვილი)
A - Action (მოქმედებისკენ მოუწოდე)
```

**PAS:**
```
P - Problem (პრობლემა)
A - Agitate (გააძლიერე)
S - Solution (გადაწყვეტა)
```

**BAB:**
```
B - Before (ახლანდელი მდგომარეობა)
A - After (სასურველი მდგომარეობა)
B - Bridge (ხიდი მათ შორის)
```
