| **DL** | ღრმა სწავლება - ML-ის ნაწილი, რომელიც იყენებს ნეირონულ ქსელებს |
| **LLM** | Large Language Model - დიდი ენობრივი მოდელი (მაგ. GPT, PaLM) |

| **Fine-tuning** | მოდელის სპეციალიზირება კონკრეტული ამოცანისთვის |
| **Hallucination** | AI-ს მიერ არასწორი ან მოგონილი ინფორმაციის გენერირება |
| **Chain of Thought** | ნაბიჯ-ნაბიჯ აზროვნების ტექნიკა AI-სთვის |

## ⚡ ChatGPT სწრაფი დაწყება

### 🔐 რეგისტრაცია (5 წუთი)
1. **chat.openai.com** → "Sign up"
2. **ელ-ფოსტა** → დადასტურება
3. **მობილური ნომერი** (ვერიფიკაციისთვის)
4. **შესვლა და სამუშაო დაწყება!**

### 🖥️ ინტერფეისის ძირითადი ელემენტები
```
┌────────────────────────────────────┐
│ [☰] ChatGPT        [⚙️] Settings  │
├────────────────────────────────────│
│                                    │
│     Chat ისტორია (მარცხნივ)       │
│                                    │
│     მთავარი Chat ფანჯარა          │
│                                    │
│     Prompt შეყვანის ველი (ქვედა)   │
└────────────────────────────────────┘
```

### 🎯 პირველი Prompt-ები (Copy & Paste)

#### ტესტირებისთვის:
```
გამარჯობა! ვარ ახალი მომხმარებელი. შეგიძლია ქართულად ახსნა
რას წარმოადგენ და რაში შეგიძლია დამხმარება?
```

#### ინფორმაციისთვის:
```
ახსენი [თემა] მარტივი ტერმინებით, გამოიყენე მაგალითები
და ბოლოს მოკლე შეჯამება
```

#### იდეებისთვის:
```
მთავაზე 5 კრეატიული იდეა [კონკრეტული ამოცანა]-სთვის.
თითოეულისთვის მოკლე ახსნა და რატომ შეიძლება იყოს ეფექტური
```

## 🛠️ Prompt Engineering - ტემპლეიტები

### 📝 ბაზისური სტრუქტურა
```
[Role/Context] + [Task] + [Format] + [Constraints]

მაგალითი:
"შენ ხარ პროფესიონალური მასწავლებელი [Role].
ახსენი ფოტოსინთეზი 10 წლის ბავშვისთვის [Task],
გამოიყენე ჩამონათვალის ფორმატი [Format],
მაქსიმუმ 200 სიტყვით [Constraint]"
```

### 🎭 Role-Based ტემპლეიტები

#### აკადემიური:
```
შენ ხარ [სფერო] ექსპერტი უნივერსიტეტში. ახსენი [თემა]
სტუდენტებისთვის, ვისაც ჯერ არ ესმით ძირითადი ცნებები.
```

#### ბიზნეს:
```
შენ ხარ გამოცდილი კონსულტანტი. შეამუშავე სტრატეგია [ამოცანა]-სთვის,
გაითვალისწინე ბიუჯეტი და დროის შეზღუდვები.
```

#### კრეატიული:
```
შენ ხარ კრეატიული დირექტორი. მოიფიქრე 3 განსხვავებული კონცეფცია
[პროექტი]-სთვის, თითოეული უნდა იყოს ინოვაციური და ორიგინალური.
```

### 🧠 Chain of Thought ტემპლეიტი
```
გადაწყვიტე ამ პრობლემა ნაბიჯ-ნაბიჯ:
1. ჯერ გაანალიზე რა არის მოცემული
2. განსაზღვრე რა უნდა ვიპოვო
3. აირჩიე შესაბამისი მეთოდი
4. შეასრულე ნაბიჯები
5. შეამოწმე პასუხი

პრობლემა: [კონკრეტული ამოცანა]
```

## 🎨 სპეციალური Prompt-ების ნაკრები

### 💼 პროფესიონალური გამოყენება

#### ელ-წერილები:
```
დაწერე პროფესიონალური ელ-წერილი [ადრესატი]-სთვის [მიზანი]-ით.
ტონი: ვეცური მაგრამ მეგობრული. შეიცავდეს: [კონკრეტული პუნქტები]
```

#### რეპორტები:
```
შექმენი ექზეკუტიური შეჯამება [თემა] რეპორტისთვის.
მთავარი პუნქტები: [1, 2, 3]. ფორმატი: Bullet points. მაქსიმუმ 300 სიტყვა.
```

#### პრეზენტაციები:
```
მოამზადე slide-ების გეგმა [თემა] პრეზენტაციისთვის [აუდიტორია]-ისათვის.
15 წუთიანი, 8-10 slide, თითოეულისთვის მიუთითე მთავარი მესიჯი.
```

### 🎓 განათლებისთვის

#### შესწავლა:
```
ახსენი [რთული ცნება] როგორც საუკეთესო მასწავლებელი.
გამოიყენე ანალოგიები, მაგალითები და ვიზუალური დახმარება.
```

#### გამოცდისთვის მომზადება:
```
შექმენი სწავლების გეგმა [გამოცდა/სერტიფიკაცია]-სთვის.
[X] კვირა დროა. გავითვალისწინე ყოველდღიური [X] საათი.
ქვეთემები: [ჩამონათვალი]
```

#### ნაშრომები:
```
დაეხმარე ესეს სტრუქტურაში [თემა]-ზე. [X] სიტყვა.
მუხდება შეიცავდეს: [მოთხოვნები]. აკადემიური სტილი, ლოგიკური არგუმენტაცია.
```

### 🎯 კრეატიული დავალებები

#### ისტორიები:
```
დაწერე [ჟანრი] ისტორია [მთავარი პერსონაჟი]-ზე.
[X] სიტყვა. ატმოსფერო: [აღწერა]. Twist ending სასურველია.
```

#### Brain Storming:
```
100 იდეა [პრობლემა]-ს გადაწყვეტისთვის. ნებისმიერი შეურაცხყოფა,
რამდენადაც უცნაური იქნება, მითუბალოი. მერე შეარჩევ ტოპ 10-ს.
```

## 🔧 troubleshooting - სწრაფი გადაწყვეტა

### 🛠️ ტექნიკური პრობლემები

| პრობლემა | გადაწყვეტა |
|----------|-------------|
| **Chat არ იტვირთება** | Browser cache გასუფთავება, VPN გამორთვა |
| **"At capacity" შეცდომა** | რამდენიმე წუთი მოელოდინე, თავიდან სცადე |
| **ქართული არ მუშაობს** | დაუყოვნებლივ მიუთითე "პასუხი ქართულად" |
| **Chat ისტორია დაიკარგა** | შემოიწმე Account settings, ისტორია შესაძლოა ღია იყოს |
| **ძალიან ნელა პასუხობს** | შემცირე prompt-ის სიგრძე, სხვა session-ი დაიწყე |

### 💡 Prompt პრობლემები

| პრობლემა | მოქმედება |
|----------|-----------|
| **ძალიან ზოგადი პასუხი** | უფრო სპეციფიკური იყავი, კონტექსტი დაამატე |
| **მარცხივ ინფორმაცია** | მიუთითე "გადამოწმე ეს ინფორმაცია" ან მოიყვანე წყაროები |
| **ძალიან გრძელი პასუხი** | მიუთითე სიტყვების რაოდენობა ან დროის შეზღუდვა |
| **არ მიყვება ინსტრუქციებს** | ახლავე დაიწყე ფორმით "ახლა უნდა..."|
| **განმეორებადი პასუხები** | მიუთითე "განსხვავებული მიდგომა" ან "ალტერნატიული პერსპექტივა" |
