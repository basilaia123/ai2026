## 🔬 ვარჯიში 2: Prompt Engineering საფუძვლები (30 წუთი)

### 🎯 მიზანი:
ეფექტური Prompt-ების შედგენის ტექნიკების ათვისება

### 📝 ვარჯიშის სტრუქტურა:

#### A ნაწილი: ზოგადი vs სპეციფიკური (10 წუთი)

**შეადარეთ ეს Prompt-ების წყვილები:**

**წყვილი 1:**
```
ზოგადი: "მითხარი ტურიზმის შესახებ"
სპეციფიკური: "ახსენი ეკოტურიზმის 5 ძირითადი პრინციპი საქართველოში,
თითოეულისთვის მოიყვანე კონკრეტული მაგალითი"
```

**წყვილი 2:**
```

სპეციფიკური: "დაწერე 500 სიტყვიანი სტატია მცირე ბიზნესის

```

**დავალება:** თითოეული წყვილისთვის ჩაწერეთ ორივე prompt და შეადარეთ შედეგები.

#### B ნაწილი: Role-Based Prompts (10 წუთი)

**შეცადეთ ამ როლებით:**

```
1. Prompt როგორც მასწავლებელი:
"შენ ხარ გამოცდილი მასწავლებელი. ახსენი ფოტოსინთეზი

2. Prompt როგორც ბიზნეს კონსულტანტი:

3. Prompt როგორც ტექნიკური ერთისი:
"შენ ხარ IT ექსპერტი. ახსენი რა არის API თავისუფადი
პროგრამისტისთვის, გამოიყენე მარტივი მაგალითები"
```

2. ცხრილის ფორმატი:
