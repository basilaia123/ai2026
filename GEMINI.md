# GEMINI.md

## Directory Overview

This directory contains the complete course materials for the **"AI - Generative Intelligence Practical Course"** (AI - გენერატიული ინტელექტის პრაქტიკაში), taught by Giorgi Basilaia at Smart Academy.

The project is a non-code repository focused on educational content. The materials are structured around 11 lectures and are primarily in Georgian, with technical terms in English. The course is designed to be practical, teaching professionals how to integrate generative AI tools into their daily workflows.

The content for each lecture is provided in multiple formats, including interactive slides, detailed summaries, study guides, and exercises.

## Key Files

*   `index.html`: The main course hub and landing page. It provides an overview of the course and links to all lecture materials. **This is the best place to start.**
*   `course-info.html`: A detailed description of the course policies, grading system (based on a 100-point scale), homework structure, and certificate requirements.
*   `CLAUDE.md`: A detailed guide for a different AI assistant. It contains an in-depth overview of the project structure, file naming conventions, content architecture, and development guidelines.
*   `Syllabus-AI-In-Practice-*.pdf`: The official syllabus for the course.
*   `lecture-X-slides.html`: The primary teaching material for each lecture (X). These are interactive, slide-based HTML presentations.
*   `lecture-X-summary.md`/`.html`: Comprehensive summaries of each lecture's key concepts.
*   `lecture-X-study-guide.md`/`.html`: Study guides for students, outlining learning objectives.
*   `lecture-X-lesson-plan.html`: Detailed lesson plans for the instructor for each lecture.
*   `homework-X.html`: Standalone HTML files containing detailed instructions for the seven homework assignments.

## Usage

The contents of this directory are intended to be viewed in a web browser.

*   To get a full overview of the course, open `index.html`.
*   To view a specific lecture, open the corresponding `lecture-X-slides.html` file.
*   The directory contains materials for both students and the instructor.
*   The file structure is consistent and well-documented in `CLAUDE.md`.
