Custom GPTs - პერსონალიზებული AI ასისტენტები

  • 🔒 Private or Public: მხოლოდ თქვენთვის ან გაზიარებადი
  • ⚡ Consistent Output: ყოველთვის იდენტური approach

  • ⚡ Trigger-Action Logic: "როცა X ხდება → გააკეთე Y"
  • 🤖 AI Integration: AI-ს აკავშირებს სხვა tools-თან

⚡ Zapier Workflows - პრაქტიკული მაგალითები

კონკრეტული automation სცენარები Gastronome.ge-ისთვის.

📧 Workflow 1: Email Summarizer

Trigger: ახალი email მოდის Gmail-ში (specific label: "Reports")
Action 1: Email content → ChatGPT → "Summarize this in 3 bullet points"
Action 2: Summary → Slack channel "daily-summaries"
Action 3: Save summary → Google Sheets log

შედეგი: ყოველი მნიშვნელოვანი email ავტომატურად summarize-დება და შეინახება!

📝 Workflow 2: Automatic Meeting Notes

Trigger: Google Calendar meeting ends
Action 1: Get meeting transcription (Google Meet / Zoom)
Action 2: Transcription → Claude → "Extract action items და key decisions"
Action 3: Results → Notion page "Meeting Notes"
Action 4: Email summary → all attendees

შედეგი: არ გჭირდება notes-ის წერა meeting-ის დროს!

📊 Workflow 3: Daily Data Reports

Trigger: Schedule (ყოველდღე 9:00 AM)
Action 1: Google Sheets → get yesterday's sales data
Action 2: Data → ChatGPT → "Analyze trends და create brief report"
Action 3: Report → Email to management team
Action 4: Save → Google Drive "Daily Reports" folder

შედეგი: ყოველდღიური რეპორტი ავტომატურად მზადდება და იგზავნება!

🛠️ First Zap Setup - Step by Step

მარტივი automation-ის დაყენება Zapier-ით.

🔴 DEMO: "Gmail → ChatGPT → Slack" Automation

სცენარიო: Customer feedback emails ავტომატურად summarize-დება და გაიგზავნება Slack-ში.

📋 Setup Steps:
  • 1. Zapier-ში რეგისტრაცია: zapier.com → Sign Up (უფასო)
  • 2. "Create Zap": ახალი automation-ის შექმნა
  • 3. Trigger Setup:
    • App: Gmail
    • Event: "New Email Matching Search"
    • Search: label:customer-feedback
    • Connect Gmail account
    • Test trigger
  • 4. Action 1 - ChatGPT:
    • App: OpenAI (ChatGPT)
    • Event: "Conversation"
    • Model: GPT-4
    • Prompt: "Summarize this customer feedback in 2-3 sentences and rate sentiment (positive/neutral/negative): {email body}"
    • Test action
  • 5. Action 2 - Slack:
    • App: Slack
    • Event: "Send Channel Message"
    • Channel: #customer-feedback
    • Message: "📧 New Feedback Summary: {ChatGPT output}"
    • Test action
  • 6. Turn On Zap: Activate → მზადაა!

💡 AI ჩვევების ჩამოყალიბება - რჩევები

✅ როგორ ჩამოვაყალიბოთ AI ჩვევები:
  • 1. Start Small: დაიწყეთ 1-2 task-ით, არა ყველაფრის ერთად შეცვლა
    მაგ: პირველ კვირაში - მხოლოდ დილის social media posts
  • 2. Same Time, Same Place: განსაზღვრული დრო და ადგილი

    რა მუშაობს? რა არა? რა გავაუმჯობესო?
