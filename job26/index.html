CV-ის სწორი სტრუქტურა

CV-ის ძირითადი კომპონენტები:

სავალდებულო სექციები:

  1. პირადი ინფორმაცია:
    • სრული სახელი და გვარი
    • ტელეფონი და ელფოსტა
    • LinkedIn პროფილი
    • ქალაქი (მისამართის მითითება საჭირო არ არის)
  2. პროფესიული შეჯამება:
    • 2-3 წინადადებიანი შესავალი
    • ძირითადი უნარების ხაზგასმა
  3. განათლება:
    • უნივერსიტეტი, ფაკულტეტი
    • სპეციალობა და კვალიფიკაცია
    • სწავლის წლები (ან მიმდინარე სტატუსი)
    • GPA (თუ 3.5-ზე მაღალია)
  4. უნარები:
    • ტექნიკური უნარები (Python, JavaScript, SQL, Data Analysis)
    • პროგრამული უზრუნველყოფა (MS Office, Adobe Creative Suite, Figma)
    • ენები (ქართული, ინგლისური, რუსული)

დამატებითი სექციები (საჭიროებისამებრ):

  • სამუშაო გამოცდილება: სტაჟირება, ნახევარ განაკვეთზე მუშაობა, ფრილანსი
  • პროექტები: საუნივერსიტეტო, პირადი, ღია კოდის (Open Source)
  • სერტიფიკატები: ონლაინ კურსები, ვორქშოპები
  • მიღწევები: ჯილდოები, ოლიმპიადები, კონკურსები
  • მოხალისეობა: არასამთავრობო ორგანიზაციებში მუშაობა
  • ინტერესები: მხოლოდ რელევანტური ინტერესები

რას არ უნდა შეიცავდეს CV:

  • ფოტო (გარდა სამოდელო და სამსახიობო სფეროებისა)
  • პირადი ინფორმაცია (ასაკი, ოჯახური მდგომარეობა)
  • ხელფასის მოლოდინი
  • სარეკომენდაციო პირების კონტაქტები ("მოთხოვნის შემთხვევაში")
  • არარელევანტური სამუშაო გამოცდილება
  • ძალიან პირადი ინტერესები

რეზიუმეს ოპტიმიზაცია: ATS და საკვანძო სიტყვები

ATS (Applicant Tracking System) ხშირად არ არის "რობოტი, რომელიც ავტომატურად გიწუნებთ" — ის უბრალოდ ახარისხებს და ფილტრავს განაცხადებს ვაკანსიაში არსებული საკვანძო სიტყვების მიხედვით.

Python · JavaScript · SQL · Excel

ინსტრუმენტები
Git · Jira · Salesforce · Power BI
ინდუსტრიის ტერმინები
Agile · Scrum · SEO · Marketing

2. დაყავი ისინი ორ ჯგუფად:

შედეგზე ორიენტირებული პუნქტები

როგორ ჩამოვაყალიბოთ ეფექტური პუნქტები?

სუსტი მაგალითი (მოვალეობების ჩამოთვლა):

  • პასუხისმგებელი ვიყავი სოციალური მედიის მართვაზე.
  • ვმონაწილეობდი მარკეტინგულ აქტივობებში.
  • ჩართული ვიყავი სხვადასხვა პროექტში.

ძლიერი მაგალითი (შედეგებზე ფოკუსირება):

  • შევქმენი და გამოვაქვეყნე 50-ზე მეტი პოსტი სოციალურ მედიაში, რამაც 3 თვეში ჩართულობა 35%-ით გაზარდა.
  • დავგეგმე და განვახორციელე Email-მარკეტინგული კამპანია 500-ზე მეტი გამომწერისთვის, რის შედეგადაც „გახსნის მაჩვენებელმა“ (Open Rate) 22%-ს მიაღწია.
  • ვხელმძღვანელობდი 4-კაციან სტუდენტურ გუნდს საუნივერსიტეტო მარკეტინგულ პროექტში, რომელმაც საუკეთესო შედეგი აჩვენა.
