# ვიზუალური პრომპტის ჩეკლისტი

AI სურათის გენერაციამდე შეამოწმეთ პრომპტი:

- [ ] **Subject:** ობიექტი ნათლად არის აღწერილი?
- [ ] **Lighting:** მითითებულია განათების ტიპი (მაგ: Golden hour)?
- [ ] **Camera:** მითითებულია რაკურსი (მაგ: Eye level, Wide)?
- [ ] **Vibe:** არის თუ არა მითითებული "Premium" ან "Minimalist"?
- [ ] **Negative Prompt:** დამატებულია აკრძალვები (No cluttered bg)?
- [ ] **Safe Space:** დატოვებულია ადგილი ტექსტისთვის (Overlay)?
- [ ] **Ratio:** არჩეულია სწორი ზომა (16:9, 9:16, 1:1)?
