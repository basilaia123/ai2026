# CRISPE და CO-STAR ჩარჩოები

CREATE-ს ალტერნატიული ფორმულები სხვადასხვა ამოცანისთვის.

## CRISPE Framework
საუკეთესოა კომპლექსური, მრავალეტაპიანი დავალებებისთვის.
- **C - Capacity:** AI-ის როლი.
- **R - Result:** რა არის საბოლოო შედეგი.
- **I - Instruction:** დეტალური ნაბიჯები.
- **S - Scope:** საზღვრები და შეზღუდვები.
- **P - Personalization:** ტონი და სტილი.
- **E - Error-Correction:** რა არ უნდა გააკეთოს AI-მ.

## CO-STAR Framework
საუკეთესოა სტრატეგიული და კრეატიული ბრიფინგისთვის.
- **C - Context:** გარემო და ფონი.
- **O - Objective:** მიზანი.
- **S - Style:** წერის სტილი.
- **T - Tone:** ემოციური განწყობა.
- **A - Audience:** სამიზნე ჯგუფი.
- **R - Response:** პასუხის ფორმატი.
