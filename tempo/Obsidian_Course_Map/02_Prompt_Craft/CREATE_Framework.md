# CREATE ფორმულა

პროფესიონალური პრომპტის აგების 6-საფეხურიანი ჩარჩო:

- **C - Context (კონტექსტი):** ვინ ვართ ჩვენ? რა არის პროექტი? (მაგ: "Tempo Holding-ის გაყიდვების გუნდი").
- **R - Role (როლი):** ვინ უნდა იყოს AI? (მაგ: "იმოქმედე როგორც უძრავი ქონების სტრატეგი").
- **E - Exact Task (ზუსტი ამოცანა):** რა არის კონკრეტული ქმედება? (მაგ: "დაწერე Email-ის დრაფტი").
- **A - Audience (აუდიტორია):** ვის ველაპარაკებით? (მაგ: "ინვესტორი, რომელიც ეძებს მაღალ ROI-ს").
- **T - Template/Format (ფორმატი):** როგორი უნდა იყოს შედეგი? (მაგ: "ცხრილი", "3 მოკლე აბზაცი").
- **E - Examples/Constraints (მაგალითები/შეზღუდვები):** რა გავითვალისწინოთ? (მაგ: "არ გამოიყენო რთული ტერმინები").
