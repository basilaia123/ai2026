# Visual Prompting Formula (S+A+E+L+C+V)

ვიზუალური პრომპტინგის ფორმულა გამოიყენება Midjourney, DALL-E 3 და Adobe Firefly-ში მაღალი ხარისხის, პროფესიონალური სურათების დასაგენერირებლად. იგი უზრუნველყოფს AI-სთვის სრული კონტექსტის მიწოდებას.

## 📌 ფორმულის კომპონენტები:
- **S - Subject (საგანი):** მთავარი ობიექტი ან პერსონაჟი, რომელიც უნდა იყოს სურათზე (მაგ. "A retro terminal box").
- **A - Active context (აქტიური კონტექსტი):** რას აკეთებს ობიექტი ან რა მდგომარეობაშია (მაგ. "glowing with green text on screen").
- **E - Environment (გარემო):** სად ხდება მოქმედება, რა არის უკანა ფონზე (მაგ. "in a dark server room").
- **L - Lighting (განათება):** რა ტიპის განათებაა გამოყენებული (მაგ. "neon rim light, soft volumetric fog").
- **C - Camera / Composition (კამერა/კომპოზიცია):** კამერის ხედვის კუთხე და მანძილი (მაგ. "close-up shot, 85mm lens, depth of field").
- **V - Vibe / Style (ვაიბი/სტილი):** სურათის საერთო განწყობა და ესთეტიკა (მაგ. "cyberpunk aesthetic, photorealistic, octane render").
